A Brief History of Hypertext

The concept of hypertext was first described by Vannevar Bush in 1945.[1] He envisioned a device called the Memex that would allow users to create trails through linked documents.

The term "hypertext" itself was coined by Ted Nelson in 1963.[2] Nelson's Project Xanadu aimed to create a universal library of interconnected documents, though it was never fully realized.[3]

Tim Berners-Lee later built on these ideas to create the World Wide Web in 1989, using a simplified form of hypertext transmitted over the internet.
