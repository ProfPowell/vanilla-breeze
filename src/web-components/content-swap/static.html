Content Swap

Both content faces are shown below. With JavaScript enabled, only one face is visible at a time and users can toggle between them.

Front

This is the front-facing content. It is displayed by default when the component is interactive.

In the enhanced version, clicking a swap trigger or the element itself reveals the back content.

Back

This is the back-facing content, normally hidden until a user toggles the swap.

It might contain additional details, a flipped card view, or secondary information.
