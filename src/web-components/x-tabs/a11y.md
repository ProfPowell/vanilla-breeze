# x-tabs Accessibility Contract

## ARIA Structure

| Element | Role | Notes |
|---------|------|-------|
| x-tab-list | tablist | Container for tabs |
| x-tab | tab | Individual tab trigger |
| x-tab-panel | tabpanel | Content panel |

## ARIA States (managed by component)

**x-tab:**
- `aria-selected="true|false"` — Current selection
- `tabindex="0|-1"` — Roving tabindex

**x-tab-panel:**
- `hidden` — Non-selected panels

## Required Properties

**x-tabs:**
- `aria-label` OR `aria-labelledby` — Describes tab group

**x-tab:**
- `aria-controls` — References panel ID (auto-generated)

**x-tab-panel:**
- `aria-labelledby` — References tab ID (auto-generated)

## Keyboard

| Key | Action |
|-----|--------|
| Tab | Enter/exit tab list |
| ArrowRight | Next tab (cycles) |
| ArrowLeft | Previous tab (cycles) |
| Home | First tab |
| End | Last tab |

## Focus Management

- Roving tabindex pattern
- Only selected tab in tab order
- Arrow keys move focus and selection together
