Project Documentation

Overview

This project uses web components built on native HTML elements. Each component degrades gracefully when JavaScript is unavailable, relying on standard <details> disclosure widgets for its baseline behavior.

Installation

Include the component script and stylesheet in your page. The <tab-set> element wraps standard <details> elements, each containing a <summary> (the tab label) and a content <div> (the panel body).

Usage

When the custom element upgrades, summaries are visually rearranged into a horizontal tab bar and panels display below. Without JavaScript, the same content appears as a stacked accordion with exclusive-open behavior provided by the shared name attribute.

API Reference

The enhanced <tab-set> component emits a tab-set:change custom event when the active tab changes, and supports data-transition for animated switching. In this static form, the browser handles all interaction natively.
