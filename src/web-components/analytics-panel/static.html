Analytics data

This site uses a first-party, cookieless analytics system. When you have JavaScript enabled and the analytics runtime is loaded, the following is captured — visible to you here and nowhere else on your device:

Pages this session
count of pages you've visited in this tab
Max scroll depth
how far down the longest page you scrolled
Active reading time
visible-tab time across this session
Queued events
component interactions waiting to flush on tab close

No cookies, no cross-site identifiers, no IP address stored. Data is cleared automatically when you close this tab.
