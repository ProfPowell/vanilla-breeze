Frequently Asked Questions

What is progressive enhancement?

Progressive enhancement is a design philosophy that starts with a baseline of functional HTML, then layers on CSS and JavaScript to improve the experience for browsers that support them.

Does this work without JavaScript?

Yes. Native <details> and <summary> elements are keyboard-accessible and togglable in every modern browser without any script.

How does single-open mode work?

The name attribute on each <details> element groups them together. The browser natively ensures only one in the group can be open at a time. The enhanced <accordion-wc> component polyfills this for older browsers.

What enhancements does the web component add?

The <accordion-wc> custom element adds richer ARIA semantics, arrow-key navigation between panels, a single-open polyfill for older browsers, and optional view-transition animations.
