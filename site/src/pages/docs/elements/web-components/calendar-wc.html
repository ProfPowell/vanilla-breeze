Overview

The <calendar-wc> component displays a month grid for viewing events, availability, or annotations on dates. Unlike <date-picker> (a form input), this is a display-focused component with optional selection for interactive UIs like booking flows.

Attributes

Attribute Type Default Description
data-month number (1-12) current Initial month to display.
data-year number current Initial year to display.
data-events JSON — Event data keyed by ISO date. Values can be a string label or { "label": "...", "color": "..." }.
data-selection string "none" Selection mode: "none", "single", or "range".
data-disabled-dates string — Comma-separated ISO dates to disable.
data-highlight-dates string — Comma-separated ISO dates to highlight, optionally with :category.

Selection Modes

Single selection

Click a date to select it. Fires calendar:select with the selected date.

Range selection

Click two dates to define a range. The range highlights between the start and end dates, with a hover preview as you move the cursor. Fires calendar:select with start and end dates.

Events

Event Detail Description
calendar:select { value, date } or { start, end, startDate, endDate } Fired when a date or range is selected.
calendar:navigate { year, month } Fired on month navigation.

Event Dots

Pass event data as JSON to show dot indicators on dates. Each date can have a string label (shown as tooltip) or an object with label and custom color.

Keyboard

Key Action
Arrow Left / Arrow Right Move focus one day.
Arrow Up / Arrow Down Move focus one week.
Enter / Space Select the focused date.

Calendar vs Date Picker

Component Purpose Form input?
<date-picker> Pick a single date for form submission Yes (ElementInternals)
<calendar-wc> Display events, availability, ranges No (use JS events)

Accessibility

The grid uses role="grid" with individual day buttons. Today is marked with aria-current="date", selected dates with aria-selected="true", and disabled dates with aria-disabled="true". The month header uses aria-live="polite" for navigation announcements. Arrow keys navigate the grid, Enter/Space select.
