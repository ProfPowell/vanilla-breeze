Overview

The <calendar-wc> component displays a month grid for viewing events, availability, or annotations on dates. Unlike <date-picker> (a form input), this is a display-focused component with optional selection for interactive UIs like booking flows.

Attributes

Attribute Type Default Description
data-month number (1-12) current Initial month to display.
data-year number current Initial year to display.
data-events JSON — Event data keyed by ISO date. Values can be a string label or { "label": "...", "color": "..." }.
data-selection string "none" Selection mode: "none", "single", or "range".
data-disabled-dates string — Comma-separated ISO dates to disable.
data-highlight-dates string — Comma-separated ISO dates to highlight, optionally with :category (holiday, deadline, event).
data-size string "default" Size variant: "compact", "default", or "large".

Size Variants

Three sizes adapt the calendar to different contexts. Large mode uses container queries for responsive square cells and shows event labels instead of dots.

Size Max Width Events Use Case
compact 16rem Dot indicators Sidebars, cards, tight spaces
default 22rem Dot indicators Standard calendar display
large 56rem Labels with icons Full-page calendars, dashboards

Selection Modes

Single selection

Click a date to select it. Fires calendar:select with the selected date.

Range selection

Click two dates to define a range. The range highlights between the start and end dates, with a hover preview as you move the cursor. Fires calendar:select with start and end dates.

Events

Event Detail Description
calendar:select { value, date } or { start, end, startDate, endDate } Fired when a date or range is selected.
calendar:navigate { year, month } Fired on month navigation.

Event Dots

Pass event data as JSON to show dot indicators on dates. Each date can have a string label (shown as tooltip) or an object with label and custom color.

Keyboard

Key Action
Arrow Left / Arrow Right Move focus one day.
Arrow Up / Arrow Down Move focus one week.
Enter / Space Select the focused date.

Calendar vs Date Picker

Component Purpose Form input?
<date-picker> Pick a single date for form submission Yes (ElementInternals)
<calendar-wc> Display events, availability, ranges No (use JS events)

Theming

The calendar exposes 27 CSS custom properties for expressive theming — from paper wall calendars to bold geometric styles. Every token defaults to transparent or the base appearance, so they are fully opt-in. Themes set them via [data-theme~="name"] calendar-wc { ... } selectors.

Banner

A decorative strip between the header and grid. Set --cal-banner-height to a non-zero value to activate. Use gradients, solid colors, SVG patterns, or photos via the image token.

TokenDefaultPurpose
--cal-banner-height0pxBanner strip height (0 hides it)
--cal-banner-bgtransparentBackground color or gradient
--cal-banner-imagenoneBackground image (photo, SVG, pattern)
--cal-banner-sizecoverBackground-size
--cal-banner-positioncenterBackground-position
[data-theme~="nature"] calendar-wc { --cal-banner-height: 5rem; --cal-banner-image: url("/images/april-landscape.jpg"); --cal-banner-size: cover; --cal-banner-position: center 40%; }

Each day button has a background token and a decoration layer (::after pseudo-element) for sticker or stamp effects. The decoration layer is behind the day number and event text.

Month-specific overrides

The component syncs data-month and data-year attributes on the host element during navigation, enabling month-specific theming.

/* Spring pastels */ [data-theme~="kawaii"] calendar-wc[data-month="3"], [data-theme~="kawaii"] calendar-wc[data-month="4"], [data-theme~="kawaii"] calendar-wc[data-month="5"] { --cal-banner-bg: linear-gradient(135deg, pink, mint); }

Accessibility

The grid uses role="grid" with individual day buttons. Today is marked with aria-current="date", selected dates with aria-selected="true", and disabled dates with aria-disabled="true". The month header uses aria-live="polite" for navigation announcements. Arrow keys navigate the grid, Enter/Space select.
