Typography Container

The layout-text element provides consistent vertical rhythm and typography styles for prose content.

Features

This component handles spacing between headings, paragraphs, lists, and other text elements automatically.

  • Consistent vertical spacing
  • Proper heading hierarchy
  • Readable line lengths

Code Example

Simply wrap your content in a <layout-text> element:

"Good typography makes content easy to read and understand."

The text container ensures all child elements have appropriate margins and spacing.
