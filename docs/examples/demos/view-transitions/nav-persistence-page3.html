Configuration

Customize the framework behavior with these configuration options:

Option Default Description
theme auto Default theme mode (light, dark, or auto)
density normal UI density (compact, normal, spacious)
transitions true Enable view transitions between pages

Each page in this demo shares the same header and sidebar structure. The browser recognizes matching view-transition-name values and morphs them in place, creating seamless navigation.
