The Magic: Full App Layout. One Attribute.

data-layout="dashboard" creates a complete app shell with sticky header, sticky sidebar, and mobile drawer animation. Toggle data-nav-open to show/hide the mobile navigation drawer.

Try it: Resize the browser below 768px (or use devtools mobile view) and click the hamburger menu to see the drawer animation. 

<div data-layout="dashboard">
  <header>Dashboard Header</header>
  <nav>Sidebar Navigation</nav>
  <main>Main Content</main>
</div>

<!-- Mobile: Toggle drawer with data-nav-open -->
