Progressive Enhancement

Vanilla Breeze follows a progressive enhancement philosophy:

  1. Tokens - CSS custom properties form the foundation
  2. Native Elements - Styled HTML5 elements work without JavaScript
  3. Custom Elements - Layout primitives use CSS-only styling
  4. Web Components - JavaScript adds interactivity on top

Each layer builds on the previous, ensuring content is accessible even when JavaScript fails.
