Attributes Reference

Complete reference for all data-* attributes used to configure layouts, components, and styling.

global Global Attributes

System-wide configuration applied to the root element.

data-mode

Used on
<html>
Values
light dark
Default
System preference

Sets the color mode for the entire page, controlling light/dark theme variants.

data-theme

Used on
<html>
Values
default ocean forest sunset modern minimal classic
Default
default

Selects the brand theme, applying different color palettes and visual styles.

data-page

Used on
<html>
Values
docs home custom
Default
none

Scopes page-specific styles using CSS @scope for layout variations.

layout Layout Attributes

Layout primitives and modifiers for building responsive page structures.

data-layout

Used on
Any block element
Values
stack cluster grid center sidebar cover switcher split prose page-stack holy-grail dashboard

Applies a layout primitive pattern. Can be used on any element as an alternative to custom elements.

data-gap

Used on
layout-stack, layout-cluster, layout-grid, layout-sidebar, layout-switcher, layout-cover, layout-reel, [data-layout]
Values
none 3xs 2xs xs s m l xl 2xl 3xl
Default
m

Controls spacing between child elements in layout containers.

data-align

Used on
layout-stack, layout-cluster, layout-split, layout-reel, [data-layout]
Values
start center end stretch baseline
Default
stretch

Controls cross-axis alignment of children in flex/grid layouts.

data-justify

Used on
layout-cluster, [data-layout="cluster"]
Values
start end center between
Default
start

Controls main-axis justification in cluster layouts.

data-min

Used on
layout-grid, layout-cover, [data-layout="grid"], [data-layout="cover"]
Values (grid)
6rem 8rem 10rem 12rem 15rem 18rem 20rem 25rem
Values (cover)
50vh 75vh 100vh 100dvh auto

Sets minimum column width for grids, or minimum height for covers.

data-max

Used on
layout-center, [data-layout="center"], [data-layout="prose"]
Values
narrow normal wide prose
Default
normal

Sets the maximum width constraint for centered content.

data-ratio

Used on
layout-split, [data-layout="split"], [data-media]
Values (split)
2:1 1:2 3:1 1:3 golden
Values (media)
1:1 4:3 3:2 16:9 21:9

Sets proportional relationships for split layouts or aspect ratios for media.

data-threshold

Used on
layout-switcher, [data-layout="switcher"]
Values
20rem 25rem 30rem 35rem 40rem 45rem
Default
30rem

Sets the width at which switcher flips between horizontal and vertical layout.

data-limit

Used on
layout-switcher, [data-layout="switcher"]
Values
2 3 4
Default
4

Maximum items per row before forcing vertical stacking.

data-side

Used on
layout-sidebar, [data-layout="sidebar"]
Values
start end
Default
start

Places the sidebar on the left (start) or right (end) side.

data-sidebar-width

Used on
layout-sidebar, [data-layout="sidebar"], [data-layout="dashboard"]
Values
narrow normal wide
Default
normal

Sets the width of the sidebar column.

data-content-min

Used on
layout-sidebar, [data-layout="sidebar"]
Values
40 50 60
Default
50

Minimum width percentage for the main content area before wrapping.

data-gutter

Used on
layout-center, [data-layout="center"]
Values
none s l
Default
s

Horizontal padding on the outside of centered content.

data-padding

Used on

none s m l xl

m

Internal padding within the container.

layout-switcher, [data-layout="switcher"]

Boolean (presence)

Values
Boolean (presence)
