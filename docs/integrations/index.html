Framework Philosophy

Vanilla Breeze follows HTML-first, CSS-second, JavaScript-third principles. This means:

  1. Components work with any backend that generates HTML
  2. No build step required for basic usage
  3. Progressive enhancement ensures content is accessible without JavaScript
  4. Web Components provide interactivity when needed
