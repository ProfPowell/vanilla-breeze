media-avatar

User profile images with fallback support and optional status indicators. Perfect for user interfaces and comment systems.

Attributes

Attribute Values Default Description
data-size xs, sm, md, lg, xl, 2xl md Avatar size
data-shape square, rounded circle Border radius style
data-ring boolean - Add border ring

Sizes

Avatars come in six sizes for different contexts.

XS SM MD LG XL 2XL
<media-avatar data-size="sm"> <span data-fallback>SM</span> </media-avatar> <media-avatar data-size="lg"> <span data-fallback>LG</span> </media-avatar>

With Image

Display a user's profile image.

<media-avatar data-size="lg"> <img src="user-photo.jpg" alt="Jane Doe" /> </media-avatar>

Fallback (Initials)

Display initials when no image is available.

JD AS MK
<media-avatar data-size="lg"> <span data-fallback>JD</span> </media-avatar>

Shapes

Choose between circle (default), rounded, or square shapes.

C R S
<media-avatar><span data-fallback>C</span></media-avatar> <media-avatar data-shape="rounded"><span data-fallback>R</span></media-avatar> <media-avatar data-shape="square"><span data-fallback>S</span></media-avatar>

Status Indicator

Show online/offline status with a colored dot.

AW BY OF
<media-avatar data-size="lg"> <img src="user-photo.jpg" alt="User Name" /> <span data-status="online"></span> </media-avatar>

Avatar Group

Display multiple avatars in an overlapping stack.

+5
<div data-avatar-group> <media-avatar><img src="user1.jpg" alt="User 1" /></media-avatar> <media-avatar><img src="user2.jpg" alt="User 2" /></media-avatar> <media-avatar><img src="user3.jpg" alt="User 3" /></media-avatar> <media-avatar><span data-fallback>+5</span></media-avatar> </div>

With Ring

Add a decorative ring around the avatar.

RG
<media-avatar data-size="lg" data-ring> <img src="user-photo.jpg" alt="User" /> </media-avatar>
