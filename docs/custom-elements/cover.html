layout-cover

Full-height container with vertically centered principal element. Perfect for hero sections, landing pages, and full-viewport layouts.

Attributes

Attribute Values Default Description
data-min auto, 50vh, 75vh, 100vh, 100dvh 100vh Minimum height of the cover
data-padding none, s, m, l, xl m Internal padding
data-gap s, m, l - Gap between non-centered elements
data-center boolean - Centers content horizontally and adds text-align center
data-nospace boolean - Removes all padding

Child Attributes

Attribute Description
data-principal Marks the element to be vertically centered
data-cover-top Pins element to the top (alternative to <header>)
data-cover-bottom Pins element to the bottom (alternative to <footer>)

Basic Usage

The cover layout vertically centers its principal element while header and footer stay at their respective edges.

Principal Content

This content is vertically centered using data-principal
Footer content

Code

<layout-cover data-min="100vh" data-padding="m" data-center> <header>Header content</header> <div data-principal> <h1>Principal Content</h1> <p>This is vertically centered</p> </div> <footer>Footer content</footer> </layout-cover>

Height Variants

The data-min attribute controls the minimum height of the cover.

data-min="auto"

Fits content
data-min="50vh"

Half viewport
data-min="75vh"

Three-quarter viewport

Code

<layout-cover data-min="auto">...</layout-cover> <layout-cover data-min="50vh">...</layout-cover> <layout-cover data-min="75vh">...</layout-cover> <layout-cover data-min="100vh">...</layout-cover> <layout-cover data-min="100dvh">...</layout-cover>

Padding Variants

The data-padding attribute controls internal spacing.

padding="none"
padding="s"
padding="l"
padding="xl"

CSS Implementation

layout-cover { display: flex; flex-direction: column; min-block-size: var(--_min-height, 100vh); padding: var(--_padding, var(--size-m)); } /* Minimum height variants */ layout-cover[data-min="50vh"] { --_min-height: 50vh; } layout-cover[data-min="75vh"] { --_min-height: 75vh; } layout-cover[data-min="100vh"] { --_min-height: 100vh; } layout-cover[data-min="100dvh"] { --_min-height: 100dvh; } layout-cover[data-min="auto"] { --_min-height: auto; } /* Padding variants */ layout-cover[data-padding="none"] { --_padding: 0; } layout-cover[data-padding="s"] { --_padding: var(--size-s); } layout-cover[data-padding="m"] { --_padding: var(--size-m); } layout-cover[data-padding="l"] { --_padding: var(--size-l); } layout-cover[data-padding="xl"] { --_padding: var(--size-xl); } /* Gap variants */ layout-cover[data-gap="s"] { gap: var(--size-s); } layout-cover[data-gap="m"] { gap: var(--size-m); } layout-cover[data-gap="l"] { gap: var(--size-l); } /* The principal (centered) element */ layout-cover > [data-principal] { margin-block: auto; } /* Header element (top-aligned) */ layout-cover > header, layout-cover > [data-cover-top] { margin-block-end: auto; } /* Footer element (bottom-aligned) */ layout-cover > footer, layout-cover > [data-cover-bottom] { margin-block-start: auto; } /* Center text content */ layout-cover[data-center] { align-items: center; text-align: center; }
