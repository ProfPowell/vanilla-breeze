layout-sidebar

Sidebar layout with a fixed-width sidebar and flexible main content area. Automatically stacks on narrow viewports.

Attributes

Attribute Values Default Description
data-side start, end start Which side the sidebar appears on
data-gap xs, s, m, l, xl m Gap between sidebar and main content
data-sidebar-width narrow, normal, wide normal Width of the sidebar (12rem, 15rem, 20rem)
data-content-min 40, 50, 60 50 Minimum width percentage for main content before wrapping
data-nowrap boolean - Prevents stacking on narrow viewports

Side Variants

The data-side attribute controls which side the sidebar appears on.

Sidebar on Start (default)

Sidebar on End

Main Content

The main content area appears first in the markup but is positioned based on flex direction.

 Sidebar

Related links, filters, or supplementary information.

Code

<!-- Sidebar on start -->
<layout-sidebar data-gap="m">
  <aside>Sidebar content</aside>
  <main>Main content</main>
</layout-sidebar>

<!-- Sidebar on end -->
<layout-sidebar data-gap="m" data-side="end">
  <main>Main content</main>
  <aside>Sidebar content</aside>
</layout-sidebar>

Width Variants

The data-sidebar-width attribute controls the sidebar width.

data-sidebar-width="narrow" (12rem)

Narrow sidebar

Main content area with more space.

data-sidebar-width="wide" (20rem)

Wide sidebar with more room for navigation or filters.

CSS Implementation

layout-sidebar {
  display: flex;
  flex-wrap: wrap;
  gap: var(--_gap, var(--size-m));
}

/* Sidebar (first child by default) */
layout-sidebar > :first-child {
  flex-basis: var(--_sidebar-width, 15rem);
  flex-grow: 1;
}

/* Main content (last child by default) */
layout-sidebar > :last-child {
  flex-basis: 0;
  flex-grow: 999;
  min-inline-size: var(--_content-min, 50%);
}

/* Gap variants */
layout-sidebar[data-gap="xs"] { --_gap: var(--size-xs); }
layout-sidebar[data-gap="s"] { --_gap: var(--size-s); }
layout-sidebar[data-gap="m"] { --_gap: var(--size-m); }
layout-sidebar[data-gap="l"] { --_gap: var(--size-l); }
layout-sidebar[data-gap="xl"] { --_gap: var(--size-xl); }

/* Sidebar on the end (right in LTR) */
layout-sidebar[data-side="end"] {
  flex-direction: row-reverse;
}

/* Sidebar width variants */
layout-sidebar[data-sidebar-width="narrow"] { --_sidebar-width: 12rem; }
layout-sidebar[data-sidebar-width="normal"] { --_sidebar-width: 15rem; }
layout-sidebar[data-sidebar-width="wide"] { --_sidebar-width: 20rem; }

/* Content minimum width variants */
layout-sidebar[data-content-min="40"] { --_content-min: 40%; }
layout-sidebar[data-content-min="50"] { --_content-min: 50%; }
layout-sidebar[data-content-min="60"] { --_content-min: 60%; }

/* No wrap - sidebar always visible */
layout-sidebar[data-nowrap] {
  flex-wrap: nowrap;
}
