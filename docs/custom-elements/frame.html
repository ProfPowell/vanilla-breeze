media-frame

Aspect ratio container for images, videos, and other media. Ensures consistent proportions regardless of content dimensions.

Attributes

Attribute Values Default Description
data-ratio 1:1, 4:3, 3:2, 16:9, 21:9, 2:3, 3:4, 9:16 16:9 Aspect ratio of the frame
data-fit cover, contain, fill, none cover How content fits within the frame (object-fit)
data-position top, bottom, left, right, center center Position of content within the frame (object-position)
data-radius s, m, l, full - Border radius of the frame

Aspect Ratio Variants

The data-ratio attribute controls the aspect ratio of the frame.

1:1 (Square) 4:3 16:9 (default) 21:9 (Ultrawide) 2:3 (Portrait) 9:16 (Story)

Code

<media-frame data-ratio="1:1"> <img src="image.jpg" alt="Square" /> </media-frame> <media-frame data-ratio="16:9"> <img src="image.jpg" alt="Widescreen" /> </media-frame> <media-frame data-ratio="2:3"> <img src="image.jpg" alt="Portrait" /> </media-frame>

Object Fit Variants

The data-fit attribute controls how content fills the frame.

data-fit="cover" (default) Fills frame, may crop data-fit="contain" Shows all content, may letterbox data-fit="fill" Stretches to fill, may distort data-fit="none" Natural size, may overflow or not fill

Object Position Variants

The data-position attribute controls the focal point when using cover fit.

data-position="top" data-position="center" data-position="bottom"

Border Radius Variants

The data-radius attribute adds rounded corners to the frame.

No radius data-radius="s" data-radius="m" data-radius="l" data-radius="full"

Understanding CSS Aspect Ratios

The aspect-ratio property makes it easy to create responsive media containers that maintain their proportions.

CSS Implementation

media-frame { display: block; position: relative; overflow: hidden; aspect-ratio: var(--_ratio, 16 / 9); } /* Common aspect ratios */ media-frame[data-ratio="1:1"] { --_ratio: 1 / 1; } media-frame[data-ratio="4:3"] { --_ratio: 4 / 3; } media-frame[data-ratio="3:2"] { --_ratio: 3 / 2; } media-frame[data-ratio="16:9"] { --_ratio: 16 / 9; } media-frame[data-ratio="21:9"] { --_ratio: 21 / 9; } media-frame[data-ratio="2:3"] { --_ratio: 2 / 3; } media-frame[data-ratio="3:4"] { --_ratio: 3 / 4; } media-frame[data-ratio="9:16"] { --_ratio: 9 / 16; } /* Media inside frame fills container */ media-frame > img, media-frame > video, media-frame > iframe, media-frame > picture, media-frame > canvas, media-frame > svg { position: absolute; inset: 0; inline-size: 100%; block-size: 100%; object-fit: var(--_fit, cover); } /* Object-fit variants */ media-frame[data-fit="cover"] { --_fit: cover; } media-frame[data-fit="contain"] { --_fit: contain; } media-frame[data-fit="fill"] { --_fit: fill; } media-frame[data-fit="none"] { --_fit: none; } /* Object position variants */ media-frame[data-position="top"] > * { object-position: top; } media-frame[data-position="bottom"] > * { object-position: bottom; } media-frame[data-position="left"] > * { object-position: left; } media-frame[data-position="right"] > * { object-position: right; } media-frame[data-position="center"] > * { object-position: center; } /* Border radius variants */ media-frame[data-radius="s"] { border-radius: var(--radius-s); } media-frame[data-radius="m"] { border-radius: var(--radius-m); } media-frame[data-radius="l"] { border-radius: var(--radius-l); } media-frame[data-radius="full"] { border-radius: var(--radius-full); }
