x-frame

Aspect ratio container for images, videos, and other media. Ensures consistent proportions regardless of content dimensions.

Attributes

Attribute Values Default Description
data-ratio 1:1, 4:3, 3:2, 16:9, 21:9, 2:3, 3:4, 9:16 16:9 Aspect ratio of the frame
data-fit cover, contain, fill, none cover How content fits within the frame (object-fit)
data-position top, bottom, left, right, center center Position of content within the frame (object-position)
data-radius s, m, l, full - Border radius of the frame

Aspect Ratio Variants

The data-ratio attribute controls the aspect ratio of the frame.

1:1 (Square) 4:3 16:9 (default) 21:9 (Ultrawide) 2:3 (Portrait) 9:16 (Story)

Code

<x-frame data-ratio="1:1">
  <img src="image.jpg" alt="Square" />
</x-frame>

<x-frame data-ratio="16:9">
  <img src="image.jpg" alt="Widescreen" />
</x-frame>

<x-frame data-ratio="2:3">
  <img src="image.jpg" alt="Portrait" />
</x-frame>

Object Fit Variants

The data-fit attribute controls how content fills the frame.

data-fit="cover" (default) Fills frame, may crop data-fit="contain" Shows all content, may letterbox data-fit="fill" Stretches to fill, may distort data-fit="none" Natural size, may overflow or not fill

Object Position Variants

The data-position attribute controls the focal point when using cover fit.

data-position="top" data-position="center" data-position="bottom"

Border Radius Variants

The data-radius attribute adds rounded corners to the frame.

No radius data-radius="s" data-radius="m" data-radius="l" data-radius="full"

Understanding CSS Aspect Ratios

The aspect-ratio property makes it easy to create responsive media containers that maintain their proportions.

CSS Implementation

x-frame {
  display: block;
  position: relative;
  overflow: hidden;
  aspect-ratio: var(--_ratio, 16 / 9);
}

/* Common aspect ratios */
x-frame[data-ratio="1:1"]   { --_ratio: 1 / 1; }
x-frame[data-ratio="4:3"]   { --_ratio: 4 / 3; }
x-frame[data-ratio="3:2"]   { --_ratio: 3 / 2; }
x-frame[data-ratio="16:9"]  { --_ratio: 16 / 9; }
x-frame[data-ratio="21:9"]  { --_ratio: 21 / 9; }
x-frame[data-ratio="2:3"]   { --_ratio: 2 / 3; }
x-frame[data-ratio="3:4"]   { --_ratio: 3 / 4; }
x-frame[data-ratio="9:16"]  { --_ratio: 9 / 16; }

/* Media inside frame fills container */
x-frame > img,
x-frame > video,
x-frame > iframe,
x-frame > picture,
x-frame > canvas,
x-frame > svg {
  position: absolute;
  inset: 0;
  inline-size: 100%;
  block-size: 100%;
  object-fit: var(--_fit, cover);
}

/* Object-fit variants */
x-frame[data-fit="cover"]   { --_fit: cover; }
x-frame[data-fit="contain"] { --_fit: contain; }
x-frame[data-fit="fill"]    { --_fit: fill; }
x-frame[data-fit="none"]    { --_fit: none; }

/* Object position variants */
x-frame[data-position="top"]    > * { object-position: top; }
x-frame[data-position="bottom"] > * { object-position: bottom; }
x-frame[data-position="left"]   > * { object-position: left; }
x-frame[data-position="right"]  > * { object-position: right; }
x-frame[data-position="center"] > * { object-position: center; }

/* Border radius variants */
x-frame[data-radius="s"] { border-radius: var(--radius-s); }
x-frame[data-radius="m"] { border-radius: var(--radius-m); }
x-frame[data-radius="l"] { border-radius: var(--radius-l); }
x-frame[data-radius="full"] { border-radius: var(--radius-full); }
