x-card

Container component with background, padding, and visual variants. The foundational surface element for grouping content.

Attributes

Attribute Values Default Description
data-variant default, elevated, outlined, ghost default Visual style variant
data-padding none, s, m, l, xl m Internal padding

Variant Styles

The data-variant attribute controls the visual appearance of the card.

Default

Default Card

Subtle shadow with background color.

data-variant="elevated"

Elevated Card

Stronger shadow for more emphasis.

data-variant="outlined"

Outlined Card

Border instead of shadow.

data-variant="ghost"

Ghost Card

Transparent, no shadow or border.

Code

<x-card>
  Default card with subtle shadow
</x-card>

<x-card data-variant="elevated">
  Elevated card with stronger shadow
</x-card>

<x-card data-variant="outlined">
  Outlined card with border
</x-card>

<x-card data-variant="ghost">
  Ghost card with no background
</x-card>

Padding Variants

The data-padding attribute controls the internal spacing.

padding="none"
padding="s" padding="m" (default) padding="l" padding="xl"

CSS Implementation

x-card {
  display: block;
  padding: var(--_padding, var(--size-m));
  background: var(--color-surface);
  border-radius: var(--radius-l);
  box-shadow: 0 1px 3px oklch(0% 0 0 / 0.1);

  &[data-variant="elevated"] {
    box-shadow:
      0 4px 6px oklch(0% 0 0 / 0.07),
      0 2px 4px oklch(0% 0 0 / 0.05);
  }

  &[data-variant="outlined"] {
    box-shadow: none;
    border: var(--border-width-thin) solid var(--color-border);
  }

  &[data-variant="ghost"] {
    box-shadow: none;
    background: transparent;
  }

  &[data-padding="none"] { --_padding: 0; }
  &[data-padding="s"]    { --_padding: var(--size-s); }
  &[data-padding="m"]    { --_padding: var(--size-m); }
  &[data-padding="l"]    { --_padding: var(--size-l); }
  &[data-padding="xl"]   { --_padding: var(--size-xl); }

  /* Nested cards get raised background */
  x-card {
    background: var(--color-surface-raised);
  }
}
