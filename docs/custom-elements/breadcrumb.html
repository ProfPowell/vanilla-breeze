nav-breadcrumb

Navigation breadcrumbs showing the user's location in a site hierarchy. Essential for wayfinding in deep site structures.

Attributes

Attribute Values Default Description
data-separator chevron, arrow, dot, pipe slash (/) Separator style between items
data-collapsed boolean - Collapse middle items for long paths

Basic Usage

Use semantic <nav> wrapping an ordered list for proper accessibility.

  1. Home
  2. Products
  3. Electronics
  4. Smartphones
<nav-breadcrumb aria-label="Breadcrumb"> <ol> <li><a href="/">Home</a></li> <li><a href="/products">Products</a></li> <li><a href="/products/electronics">Electronics</a></li> <li><span aria-current="page">Smartphones</span></li> </ol> </nav-breadcrumb>

Collapsed Breadcrumbs

For deep hierarchies, collapse middle items to save space.

  1. Home
  2. Level 1
  3. Level 2
  4. Level 3
  5. Level 4
  6. Current Page
<nav-breadcrumb data-collapsed> <ol> <li><a href="/">Home</a></li> <li><a href="/level-1">Level 1</a></li> <li><a href="/level-2">Level 2</a></li> <li><a href="/level-3">Level 3</a></li> <li><a href="/level-4">Level 4</a></li> <li><span aria-current="page">Current Page</span></li> </ol> </nav-breadcrumb>

Truncated Items

Long labels can be truncated while keeping the full text accessible via title.

  1. Home
  2. Very Long Category Name That Would Overflow
  3. Product
<li> <a href="/category" data-truncated title="Very Long Category Name"> Very Long Category Name </a> </li>

With Icons

Add icons to breadcrumb items for visual interest.

  1. Home
  2. Settings
  3. Profile
<li> <a href="/"> <icon-wc name="home"></icon-wc> Home </a> </li>

Accessibility

For proper accessibility:

  • Always include aria-label="Breadcrumb" on the nav-breadcrumb element
  • Use <ol> (ordered list) to convey hierarchy
  • Mark the current page with aria-current="page"
  • The current page should not be a link
<nav-breadcrumb aria-label="Breadcrumb"> <ol> <li><a href="/">Home</a></li> <li><a href="/products">Products</a></li> <!-- Current page: not a link, has aria-current --> <li><span aria-current="page">Smartphones</span></li> </ol> </nav-breadcrumb>
