CSS Implementation

The stack is implemented with pure CSS using flexbox:

layout-stack { display: flex; flex-direction: column; gap: var(--size-m); } layout-stack[data-gap="none"] { gap: 0; } layout-stack[data-gap="xs"] { gap: var(--size-xs); } layout-stack[data-gap="s"] { gap: var(--size-s); } layout-stack[data-gap="m"] { gap: var(--size-m); } layout-stack[data-gap="l"] { gap: var(--size-l); } layout-stack[data-gap="xl"] { gap: var(--size-xl); } layout-stack[data-align="start"] { align-items: flex-start; } layout-stack[data-align="center"] { align-items: center; } layout-stack[data-align="end"] { align-items: flex-end; } layout-stack[data-align="stretch"] { align-items: stretch; }

Key Differences: CSS-Only vs Web Component

Unlike web components like tabs-wc, layout-stack requires no JavaScript:

Feature layout-stack (CSS-only) tabs-wc (Web Component)
JavaScript required No For ARIA/keyboard
Configuration data-attributes data-attributes + ARIA
Styling CSS selectors CSS + Shadow DOM aware
Events None needed Custom events (tab-change)
