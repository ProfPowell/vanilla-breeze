nav-pagination

Page navigation for multi-page content. Supports numbered pages, prev/next controls, and various display styles.

Attributes

Attribute Values Default Description
data-variant simple, compact default Display style
data-size sm, lg medium Control size

Basic Usage

Standard pagination with page numbers and prev/next controls.

Previous 1 2 3 4 5 Next
<nav-pagination aria-label="Pagination"> <a href="/page/2" data-prev>Previous</a> <a href="/page/1">1</a> <a href="/page/2">2</a> <a href="/page/3" aria-current="page">3</a> <a href="/page/4">4</a> <a href="/page/5">5</a> <a href="/page/4" data-next>Next</a> </nav-pagination>

With Ellipsis

Use ellipsis for large page counts.

Previous 1 ... 4 5 6 ... 20 Next
<nav-pagination aria-label="Pagination"> <a href="#" data-prev>Previous</a> <a href="#">1</a> <span data-ellipsis>...</span> <a href="#">4</a> <a href="#" aria-current="page">5</a> <a href="#">6</a> <span data-ellipsis>...</span> <a href="#">20</a> <a href="#" data-next>Next</a> </nav-pagination>

Simple Variant

Only shows prev/next controls, hiding page numbers.

Previous Page 3 of 10 Next
<nav-pagination data-variant="simple" aria-label="Pagination"> <a href="#" data-prev> <icon-wc name="chevron-left"></icon-wc> Previous </a> <span data-info>Page 3 of 10</span> <a href="#" data-next> Next <icon-wc name="chevron-right"></icon-wc> </a> </nav-pagination>

Compact Variant

Connected buttons for a tighter appearance.

1 2 3 4 5
<nav-pagination data-variant="compact" aria-label="Pagination"> <a href="#" data-prev aria-label="Previous"><icon-wc name="chevron-left"></icon-wc></a> <a href="#">1</a> <a href="#">2</a> <a href="#" aria-current="page">3</a> <a href="#">4</a> <a href="#">5</a> <a href="#" data-next aria-label="Next"><icon-wc name="chevron-right"></icon-wc></a> </nav-pagination>

Sizes

Three sizes for different contexts.

Small

Prev 1 2 3 Next

Medium (default)

Prev 1 2 3 Next

Large

Prev 1 2 3 Next
<nav-pagination data-size="sm">...</nav-pagination> <nav-pagination>...</nav-pagination> <nav-pagination data-size="lg">...</nav-pagination>

Disabled States

Disable prev/next when at the first or last page.

First page (Previous disabled)

1 2 3 Next

Last page (Next disabled)

Previous 1 2 3
<!-- First page --> <button data-prev disabled>Previous</button> <!-- Last page --> <button data-next disabled>Next</button>

With Icons

Replace text labels with icons for a compact look.

1 2 3 4 5
<a href="#" data-prev aria-label="First page"> <icon-wc name="chevrons-left"></icon-wc> </a> <a href="#" data-prev aria-label="Previous page"> <icon-wc name="chevron-left"></icon-wc> </a> ... <a href="#" data-next aria-label="Next page"> <icon-wc name="chevron-right"></icon-wc> </a> <a href="#" data-next aria-label="Last page"> <icon-wc name="chevrons-right"></icon-wc> </a>

Accessibility

  • Always include aria-label="Pagination" on the nav-pagination element
  • Mark the current page with aria-current="page"
  • Use <button disabled> for disabled prev/next rather than hiding them
  • When using icon-only buttons, include aria-label on the link for screen readers
