x-cluster

Horizontal flex layout with wrapping. Ideal for groups of inline elements like buttons, tags, and navigation items.

Attributes

Attribute Values Default Description
data-gap xs, s, m, l, xl s Controls the gap between items
data-justify start, end, center, between start Controls horizontal distribution of items
data-align start, end, center, stretch, baseline center Controls vertical alignment of items
data-nowrap boolean - Prevents items from wrapping to new lines

Gap Variants

The data-gap attribute controls spacing between items.

data-gap="xs"

data-gap="m"

data-gap="xl"

Code

<x-cluster data-gap="xs">
  <button type="button">Button</button>
  <button type="button">Button</button>
  <button type="button">Button</button>
</x-cluster>

CSS Implementation

x-cluster {
  display: flex;
  flex-wrap: wrap;
  gap: var(--_gap, var(--size-s));
  align-items: var(--_align, center);
}

/* Gap variants */
x-cluster[data-gap="xs"] { --_gap: var(--size-xs); }
x-cluster[data-gap="s"] { --_gap: var(--size-s); }
x-cluster[data-gap="m"] { --_gap: var(--size-m); }
x-cluster[data-gap="l"] { --_gap: var(--size-l); }
x-cluster[data-gap="xl"] { --_gap: var(--size-xl); }

/* Justify variants */
x-cluster[data-justify="start"] { justify-content: flex-start; }
x-cluster[data-justify="end"] { justify-content: flex-end; }
x-cluster[data-justify="center"] { justify-content: center; }
x-cluster[data-justify="between"] { justify-content: space-between; }

/* Align variants */
x-cluster[data-align="start"] { --_align: flex-start; }
x-cluster[data-align="end"] { --_align: flex-end; }
x-cluster[data-align="center"] { --_align: center; }
x-cluster[data-align="stretch"] { --_align: stretch; }
x-cluster[data-align="baseline"] { --_align: baseline; }

/* No-wrap variant */
x-cluster[data-nowrap] {
  flex-wrap: nowrap;
}
