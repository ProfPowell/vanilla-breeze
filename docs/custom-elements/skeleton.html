loading-skeleton

Loading placeholder with animated pulse effect. Use to indicate content is loading while maintaining layout structure.

Attributes

Attribute Values Default Description
data-variant text, circle, rectangle text Shape of the skeleton
data-size xs, sm, md, lg, xl md Size for circle variant
data-height sm, md, lg, xl md Height for rectangle variant
data-width 25, 50, 75, 100 100 Width percentage
data-border solid, dashed, dotted - Border-only style (no fill)

Text Skeleton

Default skeleton for text content. Use multiple with varying widths to simulate paragraphs.

<layout-stack data-gap="xs"> <loading-skeleton></loading-skeleton> <loading-skeleton data-width="75"></loading-skeleton> <loading-skeleton data-width="50"></loading-skeleton> </layout-stack>

Circle Skeleton

For avatars, icons, and other circular elements.

<loading-skeleton data-variant="circle" data-size="sm"></loading-skeleton> <loading-skeleton data-variant="circle" data-size="md"></loading-skeleton> <loading-skeleton data-variant="circle" data-size="lg"></loading-skeleton>

Rectangle Skeleton

For images, cards, and larger content blocks.

<loading-skeleton data-variant="rectangle" data-height="sm"></loading-skeleton> <loading-skeleton data-variant="rectangle" data-height="md"></loading-skeleton>

Border Styles

Use data-border for outline-only placeholders without a filled background.

<loading-skeleton data-border="solid"></loading-skeleton> <loading-skeleton data-border="dashed"></loading-skeleton> <loading-skeleton data-border="dotted"></loading-skeleton> <!-- Combined with other variants --> <loading-skeleton data-variant="circle" data-size="lg" data-border="dashed"></loading-skeleton>

Accessibility

The skeleton animation respects prefers-reduced-motion. Users who prefer reduced motion will see a static placeholder instead of the pulsing animation.
