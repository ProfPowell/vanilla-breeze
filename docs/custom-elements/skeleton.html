layout-skeleton

Loading placeholder with animated pulse effect. Use to indicate content is loading while maintaining layout structure.

Attributes

Attribute Values Default Description
data-variant text, circle, rectangle text Shape of the skeleton
data-size xs, sm, md, lg, xl md Size for circle variant
data-height sm, md, lg, xl md Height for rectangle variant
data-width 25, 50, 75, 100 100 Width percentage

Text Skeleton

Default skeleton for text content. Use multiple with varying widths to simulate paragraphs.

<layout-stack data-gap="xs"> <layout-skeleton></layout-skeleton> <layout-skeleton data-width="75"></layout-skeleton> <layout-skeleton data-width="50"></layout-skeleton> </layout-stack>

Circle Skeleton

For avatars, icons, and other circular elements.

<layout-skeleton data-variant="circle" data-size="sm"></layout-skeleton> <layout-skeleton data-variant="circle" data-size="md"></layout-skeleton> <layout-skeleton data-variant="circle" data-size="lg"></layout-skeleton>

Rectangle Skeleton

For images, cards, and larger content blocks.

<layout-skeleton data-variant="rectangle" data-height="sm"></layout-skeleton> <layout-skeleton data-variant="rectangle" data-height="md"></layout-skeleton>

Accessibility

The skeleton animation respects prefers-reduced-motion. Users who prefer reduced motion will see a static placeholder instead of the pulsing animation.
