x-center

Horizontally centered content container with configurable max-width. Essential for constraining content width on wide screens.

Attributes

Attribute Values Default Description
data-max narrow, normal, wide normal Maximum width (45rem, 60rem, 80rem)
data-intrinsic boolean - Centers content that is narrower than max-width
data-text boolean - Centers text alignment
data-gutter none, s, l m Horizontal padding (gutter)

Max Width Variants

The data-max attribute controls the maximum width of the container.

data-max="narrow" (45rem) - Ideal for reading content

This content is constrained to a narrow width, perfect for long-form reading where line length matters for readability.

data-max="normal" (60rem) - Default, balanced width

Normal width provides a good balance for most content types including text, forms, and mixed layouts.

data-max="wide" (80rem) - For dashboards and data-heavy layouts

Wide containers work well for dashboards, data tables, and layouts that benefit from more horizontal space.

Code

<x-center data-max="narrow">
  <article>Long-form reading content...</article>
</x-center>

<x-center data-max="normal">
  <!-- Default width for most layouts -->
</x-center>

<x-center data-max="wide">
  <!-- Dashboard or data table -->
</x-center>

Intrinsic Centering

The data-intrinsic attribute centers content that is narrower than the max-width.

Centered Card

This card is intrinsically centered.

Code

<x-center data-intrinsic>
  <button type="button">Centered Button</button>
</x-center>

<x-center data-intrinsic>
  <x-card style="max-inline-size: 20rem;">
    Intrinsically centered card
  </x-card>
</x-center>

Text Centering

The data-text attribute adds text-align: center for centered text content.

Centered Heading

All text content within this container is centered, making it perfect for hero sections and call-to-action blocks.

Combined: Intrinsic + Text

Fully Centered

Both container and text are centered.

Code

<x-center data-text>
  <h2>Centered Heading</h2>
  <p>Centered paragraph text.</p>
</x-center>

<x-center data-intrinsic data-text>
  <!-- Both container and text centered -->
</x-center>

Gutter Variants

The data-gutter attribute controls horizontal padding (useful for nested centers).

data-gutter="none"

No side padding - content touches edges.

data-gutter="s"

Small gutter padding.

data-gutter="l"

Large gutter padding.

CSS Implementation

x-center {
  display: block;
  box-sizing: content-box;
  max-inline-size: var(--_max, var(--content-normal, 60rem));
  margin-inline: auto;
  padding-inline: var(--size-m);
}

/* Width variants */
x-center[data-max="narrow"] { --_max: var(--content-narrow, 45rem); }
x-center[data-max="normal"] { --_max: var(--content-normal, 60rem); }
x-center[data-max="wide"] { --_max: var(--content-wide, 80rem); }

/* Intrinsic centering */
x-center[data-intrinsic] {
  display: flex;
  flex-direction: column;
  align-items: center;
}

/* Text centering */
x-center[data-text] {
  text-align: center;
}

/* Gutter variants */
x-center[data-gutter="none"] {
  padding-inline: 0;
}

x-center[data-gutter="s"] {
  padding-inline: var(--size-s);
}

x-center[data-gutter="l"] {
  padding-inline: var(--size-l);
}
