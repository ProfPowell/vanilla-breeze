layout-stack

Vertical stacking layout with configurable gap. The fundamental building block for vertical rhythm.

Attributes

Attribute Values Default Description
data-gap none, 3xs, 2xs, xs, s, m, l, xl, 2xl, 3xl m Controls the vertical gap between child elements
data-align start, center, end, stretch stretch Controls horizontal alignment of child elements

Gap Variants

The data-gap attribute controls the spacing between stacked items.

data-gap="xs"

Item 1
Item 2
Item 3

data-gap="m" (default)

Item 1
Item 2
Item 3

data-gap="xl"

Item 1
Item 2
Item 3

Code

<layout-stack data-gap="xs"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> </layout-stack> <layout-stack data-gap="m"> <!-- Default gap --> </layout-stack> <layout-stack data-gap="xl"> <!-- Large gap --> </layout-stack>

Alignment Variants

The data-align attribute controls horizontal alignment of items.

data-align="start"

data-align="center"

data-align="end"

Code

<layout-stack data-gap="s" data-align="start"> <button type="button">Short</button> <button type="button">Medium Button</button> <button type="button">Longer Button Text</button> </layout-stack> <layout-stack data-gap="s" data-align="center"> <!-- Centered items --> </layout-stack> <layout-stack data-gap="s" data-align="end"> <!-- End-aligned items --> </layout-stack>

Usage Examples

Form Layout

<layout-stack data-gap="m"> <layout-stack data-gap="xs"> <label for="name">Name</label> <input type="text" id="name" /> </layout-stack> <layout-stack data-gap="xs"> <label for="email">Email</label> <input type="email" id="email" /> </layout-stack> <button type="submit">Submit</button> </layout-stack>

Card Content
