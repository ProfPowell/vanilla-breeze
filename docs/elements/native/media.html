Media

Images, video, audio, canvas, SVG, and embedded content with responsive behavior and variants.

Images

Responsive images with max-width and variant classes.

Default

Images are block-level and constrained to their container width.

<img src="image.jpg" alt="Description" loading="lazy" />

.rounded

Rounded corners for softer appearance.

<img src="image.jpg" alt="Description" class="rounded" loading="lazy" />

.circle

Circular crop for avatars. Maintains 1:1 aspect ratio.

<img src="avatar.jpg" alt="Avatar" class="circle" />

.thumbnail

Bordered thumbnail style with padding.

<img src="image.jpg" alt="Description" class="thumbnail" />

Aspect Ratio Classes

.ratio-square .ratio-video .ratio-portrait .ratio-landscape
<img src="image.jpg" alt="Square" class="ratio-square" /> <img src="image.jpg" alt="16:9" class="ratio-video" /> <img src="image.jpg" alt="Portrait" class="ratio-portrait" /> <img src="image.jpg" alt="Landscape" class="ratio-landscape" />

Object Fit Classes

.contain
.cover
<img src="image.jpg" alt="Contain" class="contain" /> <img src="image.jpg" alt="Cover" class="cover" />

Figure

Self-contained content with optional caption.

Default

<figure> <img src="image.jpg" alt="Description" /> <figcaption>Caption text here.</figcaption> </figure>

.bordered

Figure with border and padding.

<figure class="bordered"> <img src="image.jpg" alt="Description" /> <figcaption>Caption text here.</figcaption> </figure>

.centered

Centered figure and caption.

<figure class="centered"> <img src="image.jpg" alt="Description" /> <figcaption class="centered">Caption text here.</figcaption> </figure>

.float-start / .float-end

Floating figures for text wrapping.

This paragraph wraps around the floated figure. The figure floats to the inline-start (left in LTR) and text flows around it. This is useful for editorial layouts where images accompany the text content. Additional text continues to wrap until it clears the float.

A second paragraph also wraps around the figure if it is tall enough. The clearfix happens naturally when the content exceeds the figure height.

<figure class="float-start"> <img src="image.jpg" alt="Description" /> <figcaption>Caption</figcaption> </figure> <p>Text wraps around the floated figure...</p>

.code (for code examples)

<figure class="code"> <code-block language="js">const x = 1;</code-block> <figcaption>example.js</figcaption> </figure>

.quote

<figure class="quote"> <blockquote>Quote text here.</blockquote> <figcaption>Attribution</figcaption> </figure>

Caption Variants

<figcaption>Default (start aligned)</figcaption> <figcaption class="centered">Centered caption</figcaption> <figcaption class="end">End aligned caption</figcaption>

data-media (Container)

Apply data-media to any container to create an aspect ratio frame for media. Child media elements fill the container with configurable fit and position.

Attributes

Attribute Values Description
data-media (presence) Enables media container mode
data-ratio 1:1, 4:3, 3:2, 16:9, 21:9, 2:3, 3:4, 9:16 Aspect ratio (default: 16:9)
data-fit cover, contain, fill, none How media fills container (default: cover)
data-position top, bottom, left, right, center Focal point when cropping
data-radius s, m, l, full Border radius

Basic Usage

Add data-media to a container like <figure> or <div>.

<figure data-media data-ratio="16:9"> <img src="image.jpg" alt="Description" /> </figure> <figure data-media data-ratio="1:1"> <img src="image.jpg" alt="Square" /> </figure>

Object Fit

Control how media fills the container with data-fit.

cover (default) contain
<figure data-media data-ratio="1:1" data-fit="cover"> <img src="portrait.jpg" alt="..." /> </figure> <figure data-media data-ratio="1:1" data-fit="contain"> <img src="portrait.jpg" alt="..." /> </figure>

Object Position (Focal Point)

Control where the image crops from with data-position. Useful for portraits cropped to square.

top center bottom
<!-- Keep faces visible when cropping portraits --> <figure data-media data-ratio="1:1" data-position="top"> <img src="portrait.jpg" alt="Person" /> </figure>

Border Radius

<figure data-media data-ratio="1:1" data-radius="m"> <img src="avatar.jpg" alt="..." /> </figure> <figure data-media data-ratio="1:1" data-radius="full"> <img src="avatar.jpg" alt="Circular avatar" /> </figure>

Video Embeds

Works with video elements and iframes.

<div data-media data-ratio="16:9"> <iframe src="https://youtube.com/embed/..." title="Video"></iframe> </div> <div data-media data-ratio="21:9"> <video src="movie.mp4" controls></video> </div>

With Captions

Use <figure> to add captions below the media container.

<figure> <div data-media data-ratio="16:9" data-radius="m"> <img src="photo.jpg" alt="..." /> </div> <figcaption>Caption text here.</figcaption> </figure>

Variants Summary

Class Applies to Description

.circle img Circular crop (1:1 aspect)
.thumbnail img Border with padding
.bordered figure, canvas, iframe Border styling
.centered figure, figcaption Center alignment
.float-start figure Float to start (left in LTR)
.float-end figure Float to end (right in LTR)
.xs - .xxl svg Size variants
.spin svg Spinning animation
.pulse svg Pulsing animation
.embed-responsive div wrapper Responsive iframe container (deprecated, use data-media)
data-media figure, div Aspect ratio container for any media
