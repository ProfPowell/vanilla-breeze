Sectioning Elements

Semantic HTML5 sectioning elements for document structure: header, footer, nav, main, article, aside, section, and more.

Header

Introductory content for a page or section.

.site

Site-wide header with flex layout.

Site Logo
  • Home
  • About
  • Contact
<header class="site"> <a href="/">Site Logo</a> <nav class="horizontal pills" aria-label="Main navigation"> <ul> <li><a href="#" aria-current="page">Home</a></li> <li><a href="#">About</a></li> <li><a href="#">Contact</a></li> </ul> </nav> </header>

.page

Page header with bottom border.

Page Title

A brief description of this page's content.

<header class="page"> <h1>Page Title</h1> <p>A brief description of this page's content.</p> </header>

.card

Header within a card component.

Card Title

Card body content goes here.

<div class="card"> <header class="card"> <h4>Card Title</h4> </header> <div> <p>Card body content goes here.</p> </div> </div>

.sticky

Header that sticks to the top on scroll.

Sticky headers attach to the viewport top when scrolling. Best demonstrated in full page context.

<header class="site sticky"> <a href="/">Logo</a> <nav>...</nav> </header>

.centered

Center-aligned header content.

Centered Page Title

Subtitle text is also centered.

<header class="centered page"> <h1>Centered Page Title</h1> <p>Subtitle text is also centered.</p> </header>

.compact

Reduced padding for dense interfaces.

Compact Header - Less vertical space
<header class="compact"> <strong>Compact Header</strong> - Less vertical space </header>

Footer

Footer content for a page or section.

.site

Site-wide footer with background.

2024 Company Name
  • Privacy
  • Terms
  • Contact
<footer class="site"> <layout-cluster data-justify="between" data-align="center"> <span>2024 Company Name</span> <nav class="minimal" aria-label="Footer navigation"> <ul class="inline"> <li><a href="#">Privacy</a></li> <li><a href="#">Terms</a></li> </ul> </nav> </layout-cluster> </footer>

.article

Footer for article metadata.

Published on January 15, 2024 by Jane Smith

Tags: HTML, CSS, Web Development

<footer class="article"> <p>Published on January 15, 2024 by Jane Smith</p> <p>Tags: HTML, CSS, Web Development</p> </footer>

.card

Footer within a card component.

Card body content goes here.

Last updated: Today
<div class="card"> <div>Card body content</div> <footer class="card"> <layout-cluster data-justify="between"> <span>Last updated: Today</span> <button type="button" class="small">Action</button> </layout-cluster> </footer> </div>

.minimal

Simple centered footer.

2024 Your Company. All rights reserved.

<footer class="minimal"> <p>2024 Your Company. All rights reserved.</p> </footer>

.columns

Multi-column footer layout.

Company

  • About
  • Careers
  • Press

Product

  • Features
  • Pricing
  • API

Support

  • Documentation
  • Help Center
  • Contact
<footer class="columns"> <div> <h4>Company</h4> <ul class="unstyled"> <li><a href="#">About</a></li> <li><a href="#">Careers</a></li> </ul> </div> <div> <h4>Product</h4> <ul class="unstyled">...</ul> </div> <div> <h4>Support</h4> <ul class="unstyled">...</ul> </div> </footer>

Navigation landmarks with various display patterns.

.horizontal

Horizontal navigation list.

  • Home
  • Products
  • Services
  • About
  • Contact
<nav class="horizontal" aria-label="Main navigation"> <ul> <li><a href="#" aria-current="page">Home</a></li> <li><a href="#">Products</a></li> <li><a href="#">About</a></li> </ul> </nav>

.horizontal.pills

Pill-styled horizontal navigation.

  • Dashboard
  • Projects
  • Team
  • Settings
<nav class="horizontal pills" aria-label="Main navigation"> <ul> <li><a href="#" aria-current="page">Dashboard</a></li> <li><a href="#">Projects</a></li> <li><a href="#">Settings</a></li> </ul> </nav>

Tab-styled navigation with underline indicator.

  • Overview
  • Features
  • Pricing
  • Reviews
<nav class="horizontal tabs" aria-label="Product tabs"> <ul> <li><a href="#" aria-current="page">Overview</a></li> <li><a href="#">Features</a></li> <li><a href="#">Pricing</a></li> </ul> </nav>

.vertical

  • Dashboard
  • Analytics
  • Reports
  • Settings
<nav class="vertical" aria-label="Sidebar navigation"> <ul> <li><a href="#" aria-current="page">Dashboard</a></li> <li><a href="#">Analytics</a></li> <li><a href="#">Settings</a></li> </ul> </nav>

.vertical.pills

Vertical navigation with pill styling.

  • Overview

  • Billing

.breadcrumb

Breadcrumb navigation with separators.

  1. Home
  2. Products
  3. Category
  4. Current Page
<nav class="breadcrumb" aria-label="Breadcrumb"> <ol> <li><a href="#">Home</a></li> <li><a href="#">Products</a></li> <li><span aria-current="page">Current Page</span></li> </ol> </nav>

.minimal

Minimal styling for footer navigation.

  • Privacy Policy
  • Terms of Service
  • Cookie Policy
<nav class="minimal" aria-label="Legal navigation"> <ul class="inline"> <li><a href="#">Privacy Policy</a></li> <li><a href="#">Terms of Service</a></li> </ul> </nav>

Primary content area of the document.

.contained

Centered with max-width of 1200px.

Contained main content (max-width: 1200px)
<main class="contained"> <!-- Content centered with max-width: 1200px --> </main>

.narrow

Narrow width for reading content (65ch).

Narrow main for articles and blog posts (max-width: 65ch)

<main class="narrow"> <!-- Reading-optimized width (max-width: 65ch) --> </main>

.wide

Wide layout for dashboards (1600px).

Wide main for dashboards and data-heavy interfaces

<main class="wide"> <!-- Wide layout for dashboards (max-width: 1600px) --> </main>

.padded

Adds vertical padding.

Main with vertical padding for comfortable spacing

<main class="padded contained"> <!-- Combines padding with containment --> </main>

Article

Self-contained composition for syndication.

.blog

Blog-style article with centered narrow width.

Article Title

Published January 15, 2024

This is the article body content. It's constrained to a comfortable reading width.

Author: Jane Smith | Category: Technology

<article class="blog"> <header> <h1>Article Title</h1> <p>Published January 15, 2024</p> </header> <p>Article body content...</p> <footer> <p>Author: Jane Smith</p> </footer> </article>

.card

Card-styled article for listings.

Card Article

A card-style article with border and padding.

Another Card

Perfect for blog post previews or product listings.
<article class="card"> <h2>Card Article</h2> <p>A card-style article with border and padding.</p> </article>

.compact

Compact horizontal layout for lists.

Compact Article

A compact layout for list items.

Another Item

Great for notification lists or feeds.
<article class="compact"> <img src="avatar.jpg" alt="User avatar" /> <div> <h4>Compact Article</h4> <p>A compact layout for list items.</p> </div> </article>

.nested

For comments and replies with indentation.

Original Comment 2h ago

This is the original comment content.

Reply 1h ago

This is a reply to the original comment.

Nested Reply 30m ago

A reply to the reply, showing deeper nesting.

<article> <header><strong>Original Comment</strong></header> <p>Original comment content.</p> <article class="nested"> <header><strong>Reply</strong></header> <p>Reply content.</p> <article class="nested"> <header><strong>Nested Reply</strong></header> <p>Deeper nesting.</p> </article> </article> </article>

Aside

Tangentially related content like sidebars and callouts.

.sidebar

Sidebar content block.

Related Links

  • Link One
  • Link Two
  • Link Three
<aside class="sidebar"> <h4>Related Links</h4> <ul class="unstyled"> <li><a href="#">Link One</a></li> <li><a href="#">Link Two</a></li> </ul> </aside>

.callout

Highlighted information block.

Important: This callout highlights critical information that readers should pay attention to.
<aside class="callout" aria-label="Important information"> <strong>Important:</strong> Critical information here. </aside>

.note

Subtle note styling.

Note: This is additional context that may be helpful but isn't critical.
<aside class="note" aria-label="Note"> <strong>Note:</strong> Additional helpful context. </aside>

.warning

Warning callout.

Warning: Be careful when performing this action. It may have unintended consequences.
<aside class="warning" aria-label="Warning"> <strong>Warning:</strong> Be careful with this action. </aside>

.tip

Helpful tip callout.

Tip: You can speed up your workflow by using keyboard shortcuts.
<aside class="tip" aria-label="Tip"> <strong>Tip:</strong> Helpful tip content here. </aside>

.float

Floating pull-quote style.

"This aside floats to the end and text wraps around it."

This paragraph demonstrates how content flows around a floating aside. The aside is positioned to the inline-end (right in LTR) and has a maximum width of 40%. It's useful for pull quotes or supplementary information that should appear alongside the main content.

Additional paragraphs continue to wrap around the float until they clear it naturally.

<aside class="float"> <p><em>"Pull quote text here."</em></p> </aside> <p>Content that wraps around the floating aside...</p>

Vertical padding for spacing.

Padded Section

Content with comfortable vertical spacing.

.alt

Alternate background color.

Alternate Background

Useful for visual separation between page sections.

<section class="alt"> <h2>Section Title</h2> <p>Alternate background for visual separation.</p> </section>

.bordered

Top border with padding.

Bordered Section

Top border provides visual separation.

<section class="bordered"> <h2>Section Title</h2> <p>Top border provides visual separation.</p> </section>

.hero

Full-height centered hero section.

Hero Section

Centered content with minimum height.

<section class="hero"> <h1>Hero Section</h1> <p>Centered content with minimum height.</p> <button type="button">Call to Action</button> </section>

.contained

Centered with max-width.

Contained section with centered content (max-width: 1200px).

<section class="contained"> <p>Centered content (max-width: 1200px).</p> </section>

.tight

.spaced

More spacing between elements.

Main Heading

.reversed

Subtitle appears above heading.

The Beginning

.centered

Center-aligned heading group.

Centered Heading

With centered subtitle

.divided

Border between heading and subtitle.
