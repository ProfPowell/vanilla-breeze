Variants Summary

Data Attributes (Preferred)

Attribute Values Applies to Description
data-sticky header, column, both table Sticky positioning for headers or first column
data-density compact, comfortable table Cell padding density
data-align start, center, end th, td Text alignment override
data-numeric (boolean) th, td Right-align with tabular numerals
data-selected (boolean) tr Selected row highlight
data-highlight (boolean) tr Warning/attention row highlight
data-disabled (boolean) tr Disabled row state
data-responsive scroll table Horizontal scrolling on narrow viewports
