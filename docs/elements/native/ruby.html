Ruby Annotations (<ruby>)

The ruby element provides a way to display small annotations above or beside base text, commonly used for pronunciation guides in East Asian typography.

Description

The <ruby> element represents a ruby annotation, consisting of base text and ruby text (usually displayed above the base). Ruby annotations are primarily used in East Asian typography to show pronunciation or meaning of characters.

Related Elements

  • <ruby> - Container for the ruby annotation
  • <rt> - The ruby text (annotation)
  • <rp> - Ruby parentheses for fallback display in browsers without ruby support

Semantic Meaning

Ruby annotations provide pronunciation or explanatory text:

  • Furigana: Japanese pronunciation guides above kanji
  • Pinyin: Chinese romanization above hanzi
  • Bopomofo: Chinese phonetic symbols
  • Annotations: Short explanatory text for any base text

When to Use

  • Japanese text: Add furigana readings above kanji
  • Chinese text: Add pinyin or zhuyin pronunciation
  • Korean text: Add hanja readings
  • Educational content: Teach pronunciation of foreign words
  • Accessibility: Help readers with character recognition

When NOT to Use

  • Superscript: Use <sup> for mathematical superscripts
  • Footnote references: Use footnote patterns instead
  • General annotations: Use tooltips or parenthetical text for Western languages

Structure

Basic Structure

Base text followed by <rt> for the annotation.

漢字かんじ

<ruby>漢字<rt>かんじ</rt></ruby>

With Fallback Parentheses

Use <rp> to provide fallback parentheses for browsers without ruby support.

明日(あした)

<ruby>明日<rp>(</rp><rt>あした</rt><rp>)</rp></ruby>

Per-Character Annotation

Annotate each character separately for precise alignment.

漢かん字じ

<ruby>漢<rt>かん</rt>字<rt>じ</rt></ruby>

Live Examples

Japanese Furigana

日本語にほんごは美うつくしい言葉ことばです。

Translation: "Japanese is a beautiful language."

Translation: "Beijing is the capital of China."

Accessibility

  • Screen readers: May read both base text and ruby text, or be configurable
  • Fallback text: Use <rp> to provide parenthetical fallback for unsupported browsers
  • Language attribute: Set the lang attribute correctly for proper pronunciation
  • Font considerations: Ensure fonts support the characters being displayed

Best Practices

  • Always include <rp> elements for graceful degradation
  • Use appropriate lang attributes for both base and ruby text
  • Consider providing a glossary for educational content

Browser Support

Ruby annotations are well-supported in modern browsers:

  • Chrome, Edge, Safari: Full support
  • Firefox: Full support (may vary in positioning)
  • Older browsers: Fallback to inline display with <rp> parentheses

Related Elements

  • <sup> - For superscript text (different purpose)
  • <abbr> - For abbreviations with expansions
