Superscript (<sup>)

The sup element specifies inline text that should be displayed as superscript, positioned above the baseline and typically in a smaller font size.

Description

The <sup> element renders text as superscript - positioned higher than the surrounding text and typically in a smaller font. It's used for mathematical exponents, footnote references, ordinal indicators, and other typographical conventions requiring raised text.

Semantic Meaning

The superscript element indicates text that should be superscripted for semantic reasons:

  • Mathematical exponents and powers
  • Footnote and endnote references
  • Ordinal indicators (1st, 2nd, etc.)
  • Trademark and registered symbols
  • Scientific notation

Note: Use only when superscript has semantic meaning, not for purely visual effects.

When to Use

  • Mathematical exponents: x2, ex, 106
  • Footnote references: ...statement1
  • Ordinals: 1st, 2nd, 3rd
  • Powers: m2, km3
  • Chemical ions: Ca2+, SO42-

When NOT to Use

  • Visual styling: Use CSS for purely decorative positioning
  • TM/R symbols: Consider using actual Unicode characters

Variants

Default

Standard superscript positioning above the baseline.

E = mc2

E = mc<sup>2</sup>

.footnote-ref

Smaller, semi-bold styling specifically for footnote references.

This is a claim that needs citation1.

<sup class="footnote-ref"><a href="#fn1">1</a></sup>

Accessibility

  • Screen readers: May announce "superscript" but behavior varies by reader
  • Mathematical context: Readers familiar with formulas understand the notation
  • Footnotes: Ensure footnote links are keyboard accessible
  • MathML: For complex mathematics, consider using MathML instead

Accessible Footnote Pattern

Important claim1

<sup class="footnote-ref"> <a href="#fn1" id="fnref1" aria-describedby="footnotes-label">1</a> </sup>

Related Elements

sub vs sup: Quick Reference

Element Position Common Uses

<sup> Above baseline Exponents, footnotes, ordinals
