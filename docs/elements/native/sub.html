Subscript (<sub>)

The sub element specifies inline text that should be displayed as subscript, positioned below the baseline and typically in a smaller font size.

Description

The <sub> element renders text as subscript - positioned lower than the surrounding text and typically in a smaller font. It's used for typographical conventions that require subscript text, primarily in chemical formulas, mathematical expressions, and variable annotations.

Semantic Meaning

The subscript element indicates text that should be subscripted for semantic reasons:

  • Chemical formulas (atomic numbers, molecule counts)
  • Mathematical variable indices
  • Footnote numbers (in certain contexts)
  • Scientific notation

Note: Use only when subscript has semantic meaning, not for purely visual effects.

When to Use

  • Chemical formulas: H2O, CO2, C6H12O6
  • Mathematical indices: x1, x2, an
  • Variable subscripts: Vmax, Tambient
  • Phonetic transcription: Some linguistic notations

When NOT to Use

  • Visual styling: Use CSS for purely decorative positioning
  • Footnote references: Typically use <sup> for footnote numbers

Variants

Default

Standard subscript positioning below the baseline.

Water: H2O

H<sub>2</sub>O

Chemical Formulas

  • Water: H2O
  • Carbon dioxide: CO2
  • Glucose: C6H12O6
  • Sulfuric acid: H2SO4
  • Ethanol: C2H5OH
  • Methane: CH4

Mathematical Indices

The sequence x1, x2, x3, ..., xn

Matrix element: aij

Fibonacci: Fn = Fn-1 + Fn-2

Scientific Variables

Maximum velocity: Vmax = 120 km/h

Initial temperature: T0 = 25C

Partial pressure: PO2 (nested subscripts)

Combined with Superscript

Isotope notation: 14C6

Ion: Ca2+

Chemical equation: 2H2 + O2 = 2H2O

Physics Formulas

Kinetic energy: Ek = mv2/2

Electric field: Ex, Ey, Ez

In Prose

The chemical compound H2O, commonly known as water, is essential for life. Plants convert CO2 and H2O into glucose (C6H12O6) through photosynthesis.

CSS Properties

Property Value
font-size 0.75em
line-height 0 (prevents line height disruption)
position relative
vertical-align baseline

Accessibility

  • Screen readers: May announce "subscript" but behavior varies by reader
  • Scientific context: Readers familiar with formulas will understand the notation
  • Alternative text: For complex formulas, consider providing a text description
  • MathML: For complex mathematics, consider using MathML instead

Alternative for Complex Formulas

For accessibility, provide alternative descriptions for complex formulas:

C6H12O6

<span aria-label="glucose, C 6 H 12 O 6"> C<sub>6</sub>H<sub>12</sub>O<sub>6</sub> </span>

Related Elements

  • <sup> - For superscript text (exponents, footnotes)
  • <x-footnotes> - For footnote systems

sub vs sup: Quick Reference

Element Position Common Uses
<sub> Below baseline Chemical formulas, indices
<sup> Above baseline Exponents, footnotes, ordinals
