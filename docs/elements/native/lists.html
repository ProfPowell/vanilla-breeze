Unordered Lists

Default bullet-style lists for non-sequential items.

Default

  • First item in the list
  • Second item in the list
  • Third item in the list
  • Fourth item in the list
<ul> <li>First item in the list</li> <li>Second item in the list</li> <li>Third item in the list</li> <li>Fourth item in the list</li> </ul>

Nested Lists

  • Parent item one
    • Child item one
    • Child item two
    • Child item three
  • Parent item two
    • Child item one
    • Child item two
  • Parent item three
<ul> <li>Parent item one <ul> <li>Child item one</li> <li>Child item two</li> </ul> </li> <li>Parent item two</li> </ul>

.inline

Display list items horizontally with flex layout.

  • Home
  • About
  • Services
  • Contact
<ul class="inline"> <li>Home</li> <li>About</li> <li>Services</li> <li>Contact</li> </ul>

.unstyled

Remove bullets and indentation for custom styling.

  • No bullets here
  • Just plain text
  • Useful for navigation
  • Or custom layouts
<ul class="unstyled"> <li>No bullets here</li> <li>Just plain text</li> <li>Useful for navigation</li> </ul>

Ordered Lists

Numbered lists for sequential or ranked items.

Default

  1. First step in the process
  2. Second step in the process
  3. Third step in the process
  4. Fourth step in the process
<ol> <li>First step in the process</li> <li>Second step in the process</li> <li>Third step in the process</li> <li>Fourth step in the process</li> </ol>

Nested Ordered Lists

  1. Main section one
    1. Sub-section one
    2. Sub-section two
  2. Main section two
    1. Sub-section one
    2. Sub-section two
    3. Sub-section three
  3. Main section three
<ol> <li>Main section one <ol> <li>Sub-section one</li> <li>Sub-section two</li> </ol> </li> <li>Main section two</li> </ol>

Mixed Nesting

  1. Gather ingredients
    • Flour
    • Sugar
    • Eggs
  2. Mix dry ingredients
  3. Add wet ingredients
  4. Bake at 350F
<ol> <li>Gather ingredients <ul> <li>Flour</li> <li>Sugar</li> </ul> </li> <li>Mix dry ingredients</li> </ol>

.inline

  1. Step 1
  2. Step 2
  3. Step 3
  4. Step 4
<ol class="inline"> <li>Step 1</li> <li>Step 2</li> <li>Step 3</li> <li>Step 4</li> </ol>

.unstyled

  1. First item (no number)
  2. Second item (no number)
  3. Third item (no number)
<ol class="unstyled"> <li>First item (no number)</li> <li>Second item (no number)</li> <li>Third item (no number)</li> </ol>

Start Attribute

Control the starting number with the start attribute.

  1. This is item 5
  2. This is item 6
  3. This is item 7
<ol start="5"> <li>This is item 5</li> <li>This is item 6</li> <li>This is item 7</li> </ol>

Reversed Order

Use the reversed attribute for countdown lists.

  1. Bronze medal
  2. Silver medal
  3. Gold medal
<ol reversed> <li>Bronze medal</li> <li>Silver medal</li> <li>Gold medal</li> </ol>

List Types

Use the type attribute for different numbering styles.

  1. Uppercase letters
  2. Second item
  3. Third item
<ol type="A"> <li>Uppercase letters</li> <li>Second item</li> <li>Third item</li> </ol>
  1. Lowercase letters
  2. Second item
  3. Third item
<ol type="a"> <li>Lowercase letters</li> <li>Second item</li> <li>Third item</li> </ol>
  1. Roman numerals
  2. Second item
  3. Third item
<ol type="I"> <li>Roman numerals</li> <li>Second item</li> <li>Third item</li> </ol>
  1. Lowercase roman numerals
  2. Second item
  3. Third item
<ol type="i"> <li>Lowercase roman numerals</li> <li>Second item</li> <li>Third item</li> </ol>

Definition Lists

Term-description pairs for glossaries, metadata, and key-value displays.

Default

HTML
HyperText Markup Language - the standard markup language for documents designed to be displayed in a web browser.
CSS
Cascading Style Sheets - a style sheet language used for describing the presentation of a document written in HTML.
JavaScript
A programming language that enables interactive web pages and is an essential part of web applications.
<dl> <dt>HTML</dt> <dd>HyperText Markup Language</dd> <dt>CSS</dt> <dd>Cascading Style Sheets</dd> </dl>

Multiple Descriptions

A term can have multiple definitions.

Set
A collection of distinct objects.
To put something in a specified place or position.
A group of games in tennis.
Run
Move at a speed faster than walking.
An instance of operating a program.
<dl> <dt>Set</dt> <dd>A collection of distinct objects.</dd> <dd>To put something in a specified place.</dd> <dd>A group of games in tennis.</dd> </dl>
