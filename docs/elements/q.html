Inline Quote (<q>)

The q element represents a short inline quotation, automatically adding appropriate quotation marks based on the document's language.

Description

The <q> element is used for short inline quotations that don't require paragraph breaks. Browsers automatically add quotation marks, using the appropriate style for the document's language. This makes it easier to maintain consistent and correct quotation styling.

Semantic Meaning

The inline quote element conveys that the content is quoted from another source:

  • Indicates the text is a quotation, not the author's original words
  • Screen readers may announce quotations with appropriate inflection
  • Search engines understand the content is quoted
  • Quotation marks are added automatically via CSS

When to Use

  • Short quotations: Brief quotes that flow within paragraph text
  • Dialogue: When quoting what someone said
  • Technical terms: When quoting terminology from a source
  • Titles in quotes: When style requires quotes around titles
  • Scare quotes: For ironic or skeptical use of terms

When NOT to Use

  • Long quotations: Use <blockquote> for multi-paragraph quotes
  • Work titles: Use <cite> for titles of creative works
  • Code strings: Use <code> for code or technical strings

Variants

Default

Automatically adds curly quotation marks appropriate to the language.

As the saying goes, less is more.

<q>less is more</q>

Live Examples

Simple Quotation

She said, I'll be there at noon.

With Citation Link

As Dieter Rams noted, Good design is as little design as possible.

<q cite="https://example.com/rams">Good design is as little design as possible.</q>

Nested Quotations

Nested quotes automatically alternate between double and single quotation marks.

He recalled, She told me, The meeting is at three, but I forgot.

<q>She told me, <q>The meeting is at three</q>, but I forgot.</q>

In Different Languages

The lang attribute affects quotation mark style:

English: Hello, world!

French: Bonjour le monde!

German: Hallo Welt!

Japanese: こんにちは世界

<p lang="fr">French: <q>Bonjour le monde!</q></p>
<p lang="de">German: <q>Hallo Welt!</q></p>

Accessibility

  • Screen readers: May announce quotes with changed inflection or explicit "quote/unquote"
  • Visual indication: Quotation marks clearly delineate quoted content
  • Semantic clarity: Using <q> rather than typing quotes provides semantic meaning
  • Citation attribute: The cite attribute provides source information for assistive technologies

Related Elements

  • <blockquote> - For longer, block-level quotations
  • <cite> - For titles of creative works (not quotations)
