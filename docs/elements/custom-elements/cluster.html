layout-cluster

Horizontal flex layout with wrapping. Ideal for groups of inline elements like buttons, tags, and navigation items.

Attributes

Attribute Values Default Description
data-gap xs, s, m, l, xl s Controls the gap between items
data-justify start, end, center, between start Controls horizontal distribution of items
data-align start, end, center, stretch, baseline center Controls vertical alignment of items
data-nowrap boolean - Prevents items from wrapping to new lines
data-overlap xs, s, m, l, or boolean - Creates overlapping items (for avatar groups, stacked cards)

Gap Variants

The data-gap attribute controls spacing between items.

data-gap="xs"

data-gap="m"

data-gap="xl"

Code

<layout-cluster data-gap="xs"> <button type="button">Button</button> <button type="button">Button</button> <button type="button">Button</button> </layout-cluster>

Justify Variants

The data-justify attribute controls horizontal distribution.

data-justify="start" (default)

Tag 1 Tag 2 Tag 3

data-justify="center"

Tag 1 Tag 2 Tag 3

data-justify="end"

Tag 1 Tag 2 Tag 3

data-justify="between"

Start Middle End

Code

<!-- Items aligned to start (default) --> <layout-cluster data-justify="start">...</layout-cluster> <!-- Items centered horizontally --> <layout-cluster data-justify="center">...</layout-cluster> <!-- Items aligned to end --> <layout-cluster data-justify="end">...</layout-cluster> <!-- Items spread with space between --> <layout-cluster data-justify="between">...</layout-cluster>

Align Variants

The data-align attribute controls vertical alignment.

data-align="start"

data-align="center" (default)

data-align="end"

data-align="baseline"

Small Medium Large

Code

<!-- Items aligned to top --> <layout-cluster data-align="start">...</layout-cluster> <!-- Items centered vertically (default) --> <layout-cluster data-align="center">...</layout-cluster> <!-- Items aligned to bottom --> <layout-cluster data-align="end">...</layout-cluster> <!-- Items stretched to fill height --> <layout-cluster data-align="stretch">...</layout-cluster> <!-- Items aligned by text baseline --> <layout-cluster data-align="baseline">...</layout-cluster>

Overlap Variant

The data-overlap attribute creates overlapping items - useful for avatar groups and stacked card effects.

+3

Control overlap amount with size values:

data-overlap="xs" (tight)

1 2 3

data-overlap="m" (more overlap)

1 2 3
<layout-cluster data-overlap> <user-avatar><img src="user1.jpg" alt="User 1" /></user-avatar> <user-avatar><img src="user2.jpg" alt="User 2" /></user-avatar> <user-avatar><img src="user3.jpg" alt="User 3" /></user-avatar> <user-avatar><span data-fallback>+3</span></user-avatar> </layout-cluster>

CSS Implementation

layout-cluster { display: flex; flex-wrap: wrap; gap: var(--_gap, var(--size-s)); align-items: var(--_align, center); } /* Gap variants */ layout-cluster[data-gap="xs"] { --_gap: var(--size-xs); } layout-cluster[data-gap="s"] { --_gap: var(--size-s); } layout-cluster[data-gap="m"] { --_gap: var(--size-m); } layout-cluster[data-gap="l"] { --_gap: var(--size-l); } layout-cluster[data-gap="xl"] { --_gap: var(--size-xl); } /* Justify variants */ layout-cluster[data-justify="start"] { justify-content: flex-start; } layout-cluster[data-justify="end"] { justify-content: flex-end; } layout-cluster[data-justify="center"] { justify-content: center; } layout-cluster[data-justify="between"] { justify-content: space-between; } /* Align variants */ layout-cluster[data-align="start"] { --_align: flex-start; } layout-cluster[data-align="end"] { --_align: flex-end; } layout-cluster[data-align="center"] { --_align: center; } layout-cluster[data-align="stretch"] { --_align: stretch; } layout-cluster[data-align="baseline"] { --_align: baseline; } /* No-wrap variant */ layout-cluster[data-nowrap] { flex-wrap: nowrap; }
