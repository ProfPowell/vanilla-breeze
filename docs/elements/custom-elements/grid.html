layout-grid

Auto-fit responsive grid layout. Creates equal-width columns that automatically reflow based on available space.

Attributes

Attribute Values Default Description
data-min 10rem, 12rem, 15rem, 18rem, 20rem, 25rem 15rem Minimum column width before wrapping
data-gap none, xs, s, m, l, xl m Gap between grid items

Minimum Width Variants

The data-min attribute sets the minimum width each column can shrink to before the grid reflows.

data-min="10rem" (narrow)
