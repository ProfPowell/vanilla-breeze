layout-switcher

A flexbox container that automatically switches from horizontal to vertical layout based on available space. Ideal for responsive layouts without media queries.

Attributes

Attribute Values Default Description
data-threshold 20rem, 25rem, 30rem, 35rem, 40rem, 45rem 30rem Container width at which layout switches from horizontal to vertical
data-gap none, xs, s, m, l, xl m Gap between items
data-limit 2, 3, 4 - Maximum items before forcing vertical layout
data-reverse boolean - Reverses order when stacked

Basic Usage

Items display side-by-side when space allows, then stack vertically.

Item 1

Content that shares space horizontally when there's room.

Item 2

Stacks vertically when the container is narrow.

Code

<layout-switcher data-gap="m"> <div>Item 1</div> <div>Item 2</div> </layout-switcher>

Threshold Variants

The data-threshold attribute controls when the switch happens.

Lower Threshold (20rem)

Stays horizontal longer, only stacks on very narrow containers.

Item Limit

The data-limit attribute forces vertical layout when there are more items than the limit.

Code

<!-- Forces vertical when more than 2 items --> <layout-switcher data-gap="m" data-limit="2"> <div>Item 1</div> <div>Item 2</div> <div>Item 3</div> <!-- This causes stack --> </layout-switcher>

CSS Implementation

layout-switcher { display: flex; flex-wrap: wrap; gap: var(--_gap, var(--size-m)); } layout-switcher > * { flex-grow: 1; flex-basis: calc((var(--_threshold, 30rem) - 100%) * 999); } /* Threshold variants */ layout-switcher[data-threshold="20rem"] { --_threshold: 20rem; } layout-switcher[data-threshold="30rem"] { --_threshold: 30rem; } layout-switcher[data-threshold="40rem"] { --_threshold: 40rem; } /* Limit variants */ layout-switcher[data-limit="2"] > :nth-last-child(n+3), layout-switcher[data-limit="2"] > :nth-last-child(n+3) ~ * { flex-basis: 100%; }
