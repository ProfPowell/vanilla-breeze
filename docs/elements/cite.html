Citation (<cite>)

The cite element represents the title of a creative work such as a book, article, movie, song, or other referenced work.

Description

The <cite> element is used to mark up the title of a creative work. This includes books, papers, essays, poems, songs, movies, TV shows, games, sculptures, paintings, theatrical productions, operas, musicals, exhibitions, legal case reports, and more.

Semantic Meaning

The citation element specifically identifies work titles, helping:

  • Search engines understand references to creative works
  • Screen readers convey that text is a work title
  • Distinguish titles from surrounding prose visually (via italics)

Note: The <cite> element should only contain the work title, not the author name or other citation information.

When to Use

  • Book titles: The Great Gatsby, Clean Code
  • Article/paper titles: A Brief History of Time
  • Movie/TV show titles: Inception, Breaking Bad
  • Song/album titles: Bohemian Rhapsody
  • Game titles: The Legend of Zelda
  • Artwork titles: Starry Night

When NOT to Use

  • Author names: Use plain text or <span> for author attribution
  • Inline quotes: Use <q> for quoted text
  • Blockquotes: Use <blockquote> for longer quotations

Variants

Default

Displays in italic text, following typographic convention for work titles.

I recently finished reading The Elements of Typographic Style by Robert Bringhurst.

<cite>The Elements of Typographic Style</cite>

.quoted

Adds quotation marks around the title, common for shorter works like articles, chapters, or songs.

The article Why CSS Grid is Better Than Flexbox for Page Layouts explains the differences.

<cite class="quoted">Why CSS Grid is Better Than Flexbox</cite>

With Blockquote

Dieter Rams, Less and More
<blockquote> Good design is as little design as possible. <footer>Dieter Rams, <cite>Less and More</cite></footer> </blockquote>

Linked Citation

The W3C published HTML 5.2 in 2017.

<a href="https://www.w3.org/TR/html52/"><cite>HTML 5.2</cite></a>

Accessibility

  • Semantic clarity: Using <cite> helps assistive technologies understand the text represents a work title
  • Visual distinction: Italic styling helps sighted users identify titles
  • Combined with links: When linking to the work, wrap the link around the cite element for proper semantics

Related Elements

  • <blockquote> - For longer quotations (often combined with cite in footer)
  • <q> - For inline quotations from a source
  • <a> - For linking to the referenced work
