x-icon

SVG icons with Lucide integration, consistent sizing, and automatic accessibility.

Overview

The <x-icon> component provides a simple way to use SVG icons from the Lucide library or custom icon sets. Icons inherit the current text color and can be sized using preset values.

<x-icon name="home"></x-icon>
<x-icon name="settings"></x-icon>
<x-icon name="user"></x-icon>
<x-icon name="search"></x-icon>
<x-icon name="mail"></x-icon>

Attributes

Attribute Type Default Description
name string - Icon name (required). Matches filename without .svg extension.
set string "lucide" Icon set directory. Use "custom" for project-specific icons.
size string "md" Size preset: xs, sm, md, lg, xl, 2xl
label string - Accessible label for functional icons. When set, icon is announced to screen readers.
base-path string "/.assets/icons" Override the default icon path.

Sizes

Use the size attribute to control icon dimensions. Sizes are relative to the current font size.

xs sm md lg xl 2xl 
<x-icon name="star" size="xs"></x-icon>  <!-- 1em (16px at base) -->
<x-icon name="star" size="sm"></x-icon>  <!-- 1.25em (20px) -->
<x-icon name="star" size="md"></x-icon>  <!-- 1.5em (24px) -->
<x-icon name="star" size="lg"></x-icon>  <!-- 2em (32px) -->
<x-icon name="star" size="xl"></x-icon>  <!-- 2.5em (40px) -->
<x-icon name="star" size="2xl"></x-icon> <!-- 3em (48px) -->

Colors

Icons inherit currentColor from their parent element. Change the parent's color property to change the icon color.

<span style="color: var(--color-text);">
  <x-icon name="circle" size="lg"></x-icon>
</span>
<span style="color: var(--color-interactive);">
  <x-icon name="circle" size="lg"></x-icon>
</span>
<span style="color: var(--color-success);">
  <x-icon name="circle" size="lg"></x-icon>
</span>

With Buttons

Icons work naturally inside buttons. For icon-only buttons, always include an accessible label.

<!-- Icon with text label -->
<button type="button">
  <x-icon name="plus"></x-icon>
  Add Item
</button>

<!-- Icon-only button (needs label for accessibility) -->
<button type="button" class="ghost" aria-label="Settings">
  <x-icon name="settings" label="Settings"></x-icon>
</button>

The component uses the Lucide icon library by default. Lucide provides over 1,900 beautifully crafted open-source icons.

Custom Icons

Add custom icons to the .assets/icons/custom/ directory and reference them with set="custom".

<!-- Using a custom icon -->
<x-icon name="logo" set="custom"></x-icon>

<!-- Custom icon requirements:
     - SVG format with viewBox attribute
     - No fixed width/height (component handles sizing)
     - Use currentColor for strokes/fills
-->

Accessibility

Decorative vs Functional Icons

Icons can be either decorative (hidden from screen readers) or functional (announced to screen readers).

Context Pattern
Icon + visible text No label needed (decorative)
Icon-only button Add label attribute
Status indicator Add label describing status
Purely decorative No label needed

Decorative icon (with visible text):

Functional icon (icon only):

 
<!-- Decorative - icon is hidden from screen readers -->
<button>
  <x-icon name="save"></x-icon>
  Save Document
</button>

<!-- Functional - icon is announced to screen readers -->
<button aria-label="Close dialog">
  <x-icon name="x" label="Close"></x-icon>
</button>

Styling

Icons can be styled using CSS. The component uses Shadow DOM but allows color inheritance.

/* Change icon color */
.danger x-icon {
  color: var(--color-error);
}

/* Custom sizing via CSS */
x-icon.hero-icon {
  width: 4rem;
  height: 4rem;
}

/* Hover effects */
button:hover x-icon {
  color: var(--color-interactive-hover);
}

JavaScript API

The component exposes properties that mirror its attributes.

const icon = document.querySelector('x-icon');

// Read-only properties
console.log(icon.name);     // "menu"
console.log(icon.set);      // "lucide"
console.log(icon.size);     // "md"
console.log(icon.label);    // null or "Close"
console.log(icon.basePath); // "/.assets/icons"
console.log(icon.iconPath); // "/.assets/icons/lucide/menu.svg"

// Change icon dynamically
icon.setAttribute('name', 'settings');
icon.setAttribute('size', 'lg');
