tabs-wc (Enhanced Documentation Demo)

This page demonstrates the <browser-window> and <code-block> components for enhanced documentation.

Evaluation Note: This page uses two external dependencies for documentation enhancement:
  • @profpowell/code-block - Syntax highlighting with copy, line numbers
  • @profpowell/browser-window - Safari-style chrome with view source, share, download

Live Example with Browser Window

The <browser-window> component wraps live examples in Safari-style browser chrome. Using the src attribute enables view source, share, and download features.

Try it: Click the code icon in the URL bar to view source. Use the share button for more options.

Code Examples with Syntax Highlighting

The <code-block> component provides syntax highlighting, copy-to-clipboard, and line numbers.

Basic Usage

<tabs-wc aria-label="Feature tabs"> <details name="feature-tabs" open> <summary>Overview</summary> <div>Content for Overview tab</div> </details> <details name="feature-tabs"> <summary>Features</summary> <div>Content for Features tab</div> </details> <details name="feature-tabs"> <summary>Usage</summary> <div>Content for Usage tab</div> </details> </tabs-wc>

With Highlighted Lines

Draw attention to specific lines using the highlight-lines attribute:

<tabs-wc aria-label="Feature tabs"> <details name="feature-tabs" open> <summary>First Tab</summary> <div>This tab is open by default</div> </details> <details name="feature-tabs"> <summary>Second Tab</summary> <div>Click to switch to this tab</div> </details> </tabs-wc>

JavaScript Event Handling

const tabs = document.querySelector('tabs-wc'); tabs.addEventListener('tab-change', (event) => { console.log('Tab changed to:', event.detail.index); // Do something when tab changes updateContent(event.detail.index); });

Dark Theme Code Block

Code blocks support light and dark themes:

/* Style the tab list */ tabs-wc { display: block; } /* Style individual tabs */ tabs-wc summary { cursor: pointer; padding: var(--size-s) var(--size-m); } /* Active tab */ tabs-wc summary[aria-selected="true"] { border-color: var(--color-interactive); color: var(--color-interactive); } /* Tab panels */ tabs-wc details > div { padding: var(--size-m); }

Before (plain pre/code)

  <details name="tabs" open>
    <summary>Tab 1</summary>

After (code-block component)

Code Block Features

Copy to Clipboard

Every code block has a copy button (click "Copy" in the header):

npm install vanilla-breeze

Download and Share

Enable download and share buttons with attributes:

<code-block language="javascript" show-lines show-download show-share label="My Script"> // Your code here </code-block>

Collapsible Long Code

Long code blocks can be collapsed:

// This is a longer code example // that demonstrates the collapsed feature // Only the first few lines are visible // Click "Show more" to expand function example() { const items = ['one', 'two', 'three']; items.forEach(item => { console.log(item); }); return items.length; } // More code below... const result = example(); console.log('Result:', result);
