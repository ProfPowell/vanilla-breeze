Frequently Asked Questions

Everything you need to know about Vanilla Breeze.

What is Vanilla Breeze?

Vanilla Breeze is a lightweight CSS framework built on modern web standards. It uses semantic HTML, custom elements, and design tokens to help you build beautiful interfaces without a build step or JavaScript framework.

How do I get started?

Getting started is easy. You can include Vanilla Breeze via CDN with a single link tag, or install it via npm for more customization options. Check out our Getting Started guide for step-by-step instructions.

Is Vanilla Breeze free to use?

Yes, Vanilla Breeze is completely free and open source under the MIT license. You can use it in personal and commercial projects without any restrictions.

What browsers are supported?

Vanilla Breeze supports all modern browsers including Chrome, Firefox, Safari, and Edge. We target browsers with native support for CSS custom properties and custom elements (roughly the last 2 years of browser releases).

Can I use Vanilla Breeze with React or Vue?

Absolutely! While Vanilla Breeze is designed to work without a JavaScript framework, it integrates seamlessly with React, Vue, Svelte, and other frameworks. The custom elements work as native HTML elements in any environment.

How do I customize the design tokens?

Design tokens are defined as CSS custom properties, making customization straightforward. You can override any token in your own CSS, or use our theme builder tool to generate a complete custom theme with your brand colors and typography.
