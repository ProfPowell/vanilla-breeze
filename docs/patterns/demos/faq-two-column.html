Frequently Asked Questions

Quick answers to common questions about Vanilla Breeze.

What is Vanilla Breeze?

A lightweight CSS framework built on modern web standards. Semantic HTML, custom elements, and design tokens without a build step.

How do I get started?

Include Vanilla Breeze via CDN with a single link tag, or install via npm for more customization options.

Is Vanilla Breeze free to use?

Yes, it's completely free and open source under the MIT license for personal and commercial projects.

What browsers are supported?

All modern browsers including Chrome, Firefox, Safari, and Edge with native CSS custom properties support.

Can I use it with React or Vue?

Absolutely! The custom elements work as native HTML elements and integrate seamlessly with any framework.

How do I customize the design?

Override CSS custom properties in your own CSS, or use our theme builder to generate a complete custom theme.
