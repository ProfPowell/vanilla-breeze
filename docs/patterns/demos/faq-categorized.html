Getting Started

What is Vanilla Breeze?

Vanilla Breeze is a lightweight CSS framework built on modern web standards. It uses semantic HTML, custom elements, and design tokens to help you build beautiful interfaces without a build step or JavaScript framework.

How do I get started?

Include Vanilla Breeze via CDN with a single link tag, or install it via npm. Check out our Getting Started guide for step-by-step instructions.

Is Vanilla Breeze free to use?

Yes, completely free and open source under the MIT license. You can use it in personal and commercial projects without any restrictions.

Technical

What browsers are supported?

All modern browsers including Chrome, Firefox, Safari, and Edge. We target browsers with native support for CSS custom properties and custom elements.

Can I use Vanilla Breeze with React or Vue?

Absolutely! The custom elements work as native HTML elements and integrate seamlessly with React, Vue, Svelte, and other frameworks.

How do I customize the design tokens?

Design tokens are defined as CSS custom properties. Override any token in your own CSS, or use our theme builder tool to generate a complete custom theme.

Support

Where can I get help?

Visit our GitHub repository for documentation, issue tracking, and community discussions. You can also check the examples section for common patterns.

How do I report a bug?

Open an issue on our GitHub repository with a description and a minimal reproduction. We aim to triage issues within a few days.
