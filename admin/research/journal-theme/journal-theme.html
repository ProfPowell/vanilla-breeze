How it maps to Vanilla Breeze

A journaling theme isn’t a skin — it’s a stack. Each element is a surface, an edge, and a shadow, composed from the Decorated Layers family. The pieces above are all standard components wearing journal tokens.

Design approach. Everything is pure-CSS surface + one decorative layer; the only JS is the sketch-border (an SVG feDisplacementMap on a traced rect). Washi uses mix-blend-mode: multiply so it tints whatever it crosses, like real translucent tape. Imperfection is intentional — small rotate() values and jitter read as handmade. Motion is avoided by design (a journal sits still); only tabs/photos lift on hover, and that’s disabled under prefers-reduced-motion. — exploration for ProfPowell/border-wc & vanilla-breeze.
