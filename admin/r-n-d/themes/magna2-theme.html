Theme Extensions

Scrollbar · Cursor · Border Anim

Scrollbar Theming

Orange / Void · thin

The signature MAGNA scrollbar: Magnavox orange thumb on a near-void black track. The contrast is maximum — the thumb reads like a glowing phosphor strip. Scroll here to see it. More content below this line to ensure scrollability is demonstrated clearly in all browsers.

Cyan / Dark · thin

Cyan thumb on deep blue-black. The accent color variant — used for info panels, terminal outputs, and secondary content streams. The track tint (very slightly blue-black) ties the gutter to the surface. More text here to scroll.

Violet / Deep · thin

Violet thumb on deep violet-black. The secondary spectrum variant — pairs with --hue-secondary: 268. Good for modal sidebars and navigation drawers in the magna theme. The deep violet track grounds the thumb against the surface.

Orange / Void · auto width

scrollbar-width: auto — full-size native scrollbar width. Orange thumb, maximum affordance. The best a11y option: large target, maximum contrast, familiar feel. Use for primary content areas where scrolling must be immediately obvious.

scrollbar-width: none — hidden (still scrollable)

No visible scrollbar. Container still scrolls via trackpad, touch, or keyboard. The content is accessible — only the chrome is hidden. Useful for carousels, horizontal scroll-snapping, and fullscreen immersive experiences. More text to scroll here.

Token architecture

/* In :root[data-theme~="magna"] */
--scrollbar-thumb:
  oklch(68% 0.22 55);
--scrollbar-track:
  oklch(8% 0.02 255);
--scrollbar-width: thin;
/* Applied to :root globally */

Cursor Theming

Default cursor

Orange crosshair SVG
--cursor-custom-default

Orange arrow SVG
--cursor-custom-pointer

SVG cursors are inline data URIs — zero network requests. Colors are in hex (SVG fill doesn't accept OKLCH in cursor URLs). Fallback to crosshair / pointer on unsupported browsers.
