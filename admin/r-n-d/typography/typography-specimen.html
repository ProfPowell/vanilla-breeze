Typography specimen: no CSS vs old CSS vs modern CSS

A single prose sample displayed three ways so students can compare raw HTML defaults, older conventional article styling, and a more modern fall-forward CSS typography approach.

What to inspect

Look at measure, line-height, heading breaks, link underlines, blockquote treatment, list spacing, code styling, and how a long URL behaves. The modern mode uses features like clamp(), max-inline-size: 68ch, text-wrap: balance, text-wrap: pretty, font-optical-sizing: auto, logical properties, and underline tuning.

The “old CSS” view is deliberately respectable but dated: fixed-ish widths, simpler heading scale, ordinary wrapping, plain underlines, and fewer typographic niceties.
