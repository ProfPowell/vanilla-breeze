# Upcoming features in browsers

Vanilla Breeze is forward thinking and as such we aim to use the latest and greatest features in browsers. This includes embracing new APIs, CSS properties, and JavaScript features that enhance performance, security, and user experience and may not be available in baseline or even all browsers.  If Chrome has it we are likely interested in it.

- Chrome 147
  - Element Scoped View Transitions
  	Should allow us to do more interesting transition effects particular in lists.
