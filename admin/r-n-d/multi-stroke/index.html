Multi-stroke text

Eight standalone demos exploring the stacked-text-stroke trick from yuanchuan's article. Vanilla HTML / CSS / JS — no build step. Open any file directly in a browser.

  1. 01 / basics

    The stack

    Hardcoded layers, alternating widths and colors. The whole technique in one screen.

    CSS only
  2. 02 / typography

    Kanji 愛 · 秋收冬藏

    Same trick, Japanese characters and vertical writing. Glyph shape changes everything.

    CSS only
  3. 03 / variables

    Re-skin via tokens

    One markup, three skins. --c1, --c2, --base drive everything.

    CSS only
  4. 04 / comic

    POW! burst

    Halftone dots, radiating ink rays, tilted multi-stroke. Pop-comic palette.

    CSS only
  5. 05 / animation

    Pulse & ripple

    @property registers --base and --c so they interpolate. Phase delay → ripple.

    CSS only
  6. 06 / interactive

    Live controls

    Sliders, color pickers, font select. JS just writes CSS variables.

    JS
  7. 07 / construction

    Click to build

    Each click prepends an outer ring. Watch the effect emerge layer by layer.

    JS
  8. 08 / showpiece

    SuperGraphics poster

    Mouse-driven hue and base-width, spinning bands, halftone, EN ↔ 夢 toggle.

    JS

Note: -webkit-text-stroke is well-supported but performance degrades at large font sizes (the article calls this out). Fine for experiments and posters; less great for body copy.
