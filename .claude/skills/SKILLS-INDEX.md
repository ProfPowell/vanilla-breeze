# Skills Index

Quick reference for all project skills. **Consult before starting work.**

## By File Type

| Extension | Skill | Description |
|-----------|-------|-------------|
| `.html`, `.xhtml`, `.htm` | xhtml-author | Valid XHTML-strict HTML5 markup |
| `.css` | css-author | Modern CSS with @layer, nesting, tokens |
| `.js`, `.mjs` | javascript-author | Vanilla JS for Web Components |
| `.ts`, `.tsx`, `.mts`, `.cts` | typescript-author | Strict TypeScript patterns |
| `.md` | markdown-author | Quality markdown content |
| `.astro` | astro | Astro content sites |
| `.njk`, `.liquid` | eleventy | Eleventy (11ty) templates |
| `Dockerfile*` | containerization | Docker containers |
| `docker-compose*.yml` | containerization | Docker Compose configuration |
| `*.config.js`, `vite.config.*` | build-tooling | Vite and build configs |
| `tsconfig*.json` | typescript-author | TypeScript configuration |
| `.env*` | env-config | Environment variables |
| `.github/workflows/*.yml` | ci-cd | GitHub Actions |
| `migrations/*.js` | database | Database migrations |
| `bin/*.js` | cli-author | CLI entry points |
