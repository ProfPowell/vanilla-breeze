Layout Patterns

Foundational layout utilities for building consistent, responsive page layouts.

Content Width Variants
data-width="narrow"

Best for long-form prose content like blog posts and articles. Maximum width of 40rem.
data-width="default"

Standard content width suitable for most pages. Maximum width of 65rem.
data-width="wide"

Extra wide for dashboards, data tables, and grid-heavy layouts. Maximum width of 80rem.

Full Bleed Sections

Break out of content containers to span the full viewport width.
data-bg="surface"

Surface background that spans full width while content stays centered.
data-bg="accent"

Accent color background for highlighting important sections.
data-bg="dark"

Dark background for contrast and visual separation.
