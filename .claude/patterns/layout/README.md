# Layout Patterns

Structural page organization.

## Components

- `page-shell` - Basic page wrapper
- `content-width` - Centered content
- `full-bleed` - Edge-to-edge section
- `split-layout` - Two-column
- `sidebar-layout` - Main + sidebar
- `dashboard-layout` - App shell
- `card-grid` - Responsive card grid
- `masonry` - Pinterest-style
- `stack` - Vertical spacing
- `cluster` - Horizontal grouping
- `divider` - Section separator
- `spacer` - Vertical space

## Guidelines

- Layouts use CSS Grid and Flexbox
- Fluid spacing with `clamp()` and design tokens
- Container queries for component-level responsiveness
- Named grid areas for page structure
