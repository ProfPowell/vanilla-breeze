Button Component

Primary interactive element for actions. Uses data-button attribute for variants.

Variants

Four semantic variants for different action types.

As Links

Primary Link Secondary Link Ghost Link Danger Link

Sizes

Three sizes using data-size attribute.

With Icons

Buttons can include icons alongside text.

Icon Only

States

Disabled

Link Disabled

Loading

Full Width

Use data-full-width for form submit buttons.

    •

    Usage

    <!-- Primary button -->
<button type="button" data-button="primary">Submit</button>

<!-- Secondary with icon -->
<button type="button" data-button="secondary">
  <x-icon name="plus"></x-icon>
  Add Item
</button>

<!-- Link styled as button -->
<a href="/action" data-button="primary">Continue</a>

<!-- Small danger button -->
<button type="button" data-button="danger" data-size="small">Delete</button>

<!-- Icon-only button -->
<button type="button" data-button="ghost" data-icon-only aria-label="Settings">
  <x-icon name="settings"></x-icon>
</button>
