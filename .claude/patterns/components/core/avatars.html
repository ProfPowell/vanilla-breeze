Avatar Component

User representations with images or initials. Uses <user-avatar> custom element.

Sizes

Five sizes from extra small to extra large.

XS SM MD LG XL

With Images

Initials

When no image is available, display user initials.

JD AK SM RB TC

Shapes

Circle (default) or square with data-shape.

Status Indicators

Show online status with data-status.

With Initials

JD AK SM

Avatar Groups

Stack avatars for team displays.

+5

In Context

User List Item

Comment

Usage

<!-- With image -->
<user-avatar>
  <img src="/avatar.jpg" alt="Jane Doe" />
</user-avatar>

<!-- With initials -->
<user-avatar>JD</user-avatar>

<!-- Large with status -->
<user-avatar data-size="lg" data-status="online">
  <img src="/avatar.jpg" alt="Jane Doe" />
</user-avatar>

<!-- Square shape -->
<user-avatar data-shape="square">
  <img src="/team-logo.jpg" alt="Team" />
</user-avatar>
