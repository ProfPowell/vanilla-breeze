Icon Component

Consistent, accessible icons using Lucide library with the <x-icon> Web Component.

Sizes

Icons scale with their context using preset sizes.

  • xs (16px)
  • sm (20px)
  • md (24px)
  • lg (32px)
  • xl (40px)
  • 2xl (48px)

Accessibility

  • Decorative icons: No label needed when text is present
  • Functional icons: Add label attribute for icon-only buttons
  • Icons inherit currentColor for consistent theming
  • Use appropriate size for touch targets (minimum 44x44px for mobile)

Usage

<!-- Basic icon -->
<x-icon name="star"></x-icon>

<!-- With size -->
<x-icon name="home" size="lg"></x-icon>

<!-- In button (decorative - no label needed) -->
<button data-button="primary">
  <x-icon name="plus" size="sm"></x-icon>
  Add Item
</button>

<!-- Icon-only button (needs label for accessibility) -->
<button data-button="ghost" aria-label="Close">
  <x-icon name="x" label="Close"></x-icon>
</button>

<!-- Custom icon set -->
<x-icon name="logo" set="custom"></x-icon>
