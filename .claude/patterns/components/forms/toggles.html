Accessibility

  • Use role="switch" for proper screen reader announcement
  • The label wraps the entire toggle for larger click target
  • Focus ring appears on keyboard navigation
  • Disabled state prevents interaction and reduces opacity

Usage

<form-field>
  <label>
    <input type="checkbox" name="setting" role="switch"/>
    <toggle-track>
      <toggle-thumb/>
    </toggle-track>
    <span>Enable feature</span>
  </label>
</form-field>

<!-- Checked by default -->
<form-field>
  <label>
    <input type="checkbox" name="active" role="switch" checked="checked"/>
    <toggle-track>
      <toggle-thumb/>
    </toggle-track>
    <span>Active</span>
  </label>
</form-field>
