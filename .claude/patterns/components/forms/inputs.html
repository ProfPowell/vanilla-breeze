Input Component

Text inputs wrapped in <form-field> for consistent labeling and validation.

Input Types

With Help Text

Use <output> for dynamic help/error messages.

3-20 characters, letters and numbers only Must be at least 8 characters

Validation States

CSS-only validation using :user-valid and :user-invalid.

Enter a valid email address This field is required Only letters allowed

States

Textarea

Maximum 500 characters At least 10 characters

Select

Usage

<!-- Basic input -->
<form-field>
  <label for="name">Name</label>
  <input type="text" id="name" name="name" placeholder="John Doe"/>
</form-field>

<!-- With validation message -->
<form-field>
  <label for="email">Email</label>
  <input type="email" id="email" name="email" required="required"
         aria-describedby="email-msg"/>
  <output id="email-msg" for="email">Enter a valid email</output>
</form-field>

<!-- Select -->
<form-field>
  <label for="country">Country</label>
  <select id="country" name="country">
    <option value="us">United States</option>
  </select>
</form-field>

<!-- Textarea -->
<form-field>
  <label for="message">Message</label>
  <textarea id="message" name="message" rows="4"></textarea>
</form-field>
