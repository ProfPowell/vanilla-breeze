Meter Component

Visual indicator for scalar values within a known range. Uses the native <meter> element.

Basic Meter

The meter element displays a value within a range using automatic color coding.

Meter vs Progress

Choose the right element for your use case:

Use Element Example
Task completion <progress> File upload
Scalar measurement <meter> Disk usage
Loading state <progress> Page loading
Quality rating <meter> Signal strength

Usage

<!-- Basic meter -->
<meter-display>
  <label>Disk Usage</label>
  <meter value="0.6" min="0" max="1">60%</meter>
  <span data-value>60%</span>
</meter-display>

<!-- With value ranges -->
<meter value="0.8"
       min="0"
       max="1"
       low="0.2"
       high="0.8"
       optimum="0.9">
  80%
</meter>

<!-- Inverted (lower is better) -->
<meter value="0.02"
       min="0"
       max="1"
       low="0.05"
       high="0.15"
       optimum="0">
  2% error rate
</meter>
