Accessibility

  • Uses native <details> element for built-in accessibility
  • The name attribute creates exclusive accordion behavior
  • Only one panel can be open at a time (like tabs)
  • Keyboard accessible: Enter/Space to toggle, arrow keys between items
  • No JavaScript required for core functionality

Usage

<!-- Basic accordion (default) -->
<tab-group>
  <details name="unique-name" open>
    <summary>Tab 1</summary>
    <p>Content for tab 1</p>
  </details>
  <details name="unique-name">
    <summary>Tab 2</summary>
    <p>Content for tab 2</p>
  </details>
</tab-group>

<!-- Pills style -->
<tab-group data-style="pills">
  ...
</tab-group>

<!-- Key: The name attribute must be unique per tab group
     but shared among all details in that group -->

How It Works

The name attribute on <details> elements creates an "exclusive accordion" where only one can be open at a time - exactly like tabs but with native HTML.

  • No JavaScript - Browser handles open/close state
  • Vertical or horizontal - CSS controls layout via data-style
  • Progressive enhancement - Works in all browsers that support details
