Accessibility

  • Use aria-label on the nav-pagination to describe its purpose
  • Use aria-current="page" to indicate the current page
  • Disabled states should use the disabled attribute on buttons
  • All links should have meaningful href values

Usage

<nav-pagination aria-label="Page navigation">
  <nav>
    <a href="?page=2" data-page="prev">Previous</a>
    <a href="?page=1">1</a>
    <a href="?page=2">2</a>
    <a href="?page=3" aria-current="page">3</a>
    <a href="?page=4">4</a>
    <a href="?page=5">5</a>
    <a href="?page=4" data-page="next">Next</a>
  </nav>
</nav-pagination>

<!-- Disabled button for first/last page -->
<button type="button" disabled>Previous</button>
