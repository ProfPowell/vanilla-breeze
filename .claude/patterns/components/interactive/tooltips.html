Tooltip Component

Contextual hints that appear on hover/focus. Uses the <tool-tip> Web Component with native Popover API.

Basic Tooltips

Hover or focus on buttons to see tooltips. The component handles hover/focus automatically.

Positions

Tooltips can appear in different positions relative to their trigger.

Rich Content Tooltips

Use a <template data-tooltip> for HTML content.

Icon Button Tooltips

Particularly useful for icon-only buttons to provide context.

In Context

Toolbar

Form Field Help

Accessibility

  • Component automatically sets aria-describedby on trigger
  • Tooltips appear on both hover and focus
  • Escape key dismisses tooltip (native Popover API)
  • Keep tooltip text concise and informative
  • Don't put essential information only in tooltips

Usage

<!-- Simple text tooltip -->
<tool-tip content="Save your changes">
  <button>Save</button>
</tool-tip>

<!-- With position -->
<tool-tip content="Appears above" position="top">
  <button>Hover me</button>
</tool-tip>

<!-- Rich content tooltip (use template for HTML) -->
<tool-tip position="top">
  <button>Hover me</button>
  <template data-tooltip>
    <strong>Formatted</strong> content with <kbd>Ctrl+S</kbd>
  </template>
</tool-tip>

<!-- Icon button with tooltip -->
<tool-tip content="Edit">
  <button aria-label="Edit">
    <icon-wc name="pencil"></icon-wc>
  </button>
</tool-tip>

Attributes

Attribute Default Description
content - Simple text tooltip content
position top Position: top, bottom, left, right
data-tooltip-delay 200 Show delay in milliseconds
