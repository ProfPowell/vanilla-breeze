# Navigation Patterns

Wayfinding and page structure elements.

## Components

- `site-header` - Primary navigation bar
- `site-footer` - Footer with links
- `nav-menu` - Navigation list
- `breadcrumbs` - Location trail
- `tabs` - Tabbed interface
- `sidebar` - Side navigation
- `pagination` - Page navigation
- `steps` - Progress steps
- `toc` - Table of contents
- `skip-link` - Accessibility skip

## Guidelines

- Navigation uses semantic `<nav>` elements
- Current page indicated with `aria-current="page"`
- Keyboard navigation supported
- Mobile-responsive with hamburger or drawer patterns
