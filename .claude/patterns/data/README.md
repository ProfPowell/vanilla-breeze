# Data Patterns

Structured data display and interaction.

## Components

- `data-table` - Tabular data
- `data-list` - List of records
- `definition-list` - Key-value pairs
- `key-value` - Single pair
- `comparison-table` - Feature comparison
- `tree-view` - Hierarchical data
- `calendar` - Date grid
- `chart-wrapper` - Chart container

## Guidelines

- Tables use `<table-wrapper>` for responsive behavior
- Large datasets support virtual scrolling
- Sortable columns indicated with ARIA
- Charts wrapped in `<figure>` with caption
