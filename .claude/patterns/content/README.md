# Content Patterns

Display and presentation of information.

## Components

- `hero` - Large header section
- `feature-grid` - Feature highlights
- `card` - Content container
- `product-card` - Product display
- `blog-card` - Blog post preview
- `testimonial` - Customer quote
- `pricing-table` - Price comparison
- `team-grid` - Team members
- `timeline` - Chronological events
- `stats` - Key metrics
- `gallery` - Image collection
- `faq-list` - Q&A accordion
- `press-list` - Press releases
- `logo-cloud` - Partner/client logos

## Guidelines

- Content patterns use semantic HTML (`<article>`, `<section>`, `<figure>`)
- Images include responsive `srcset` and `sizes`
- Text uses proper heading hierarchy
- Structured data (JSON-LD) included where applicable
