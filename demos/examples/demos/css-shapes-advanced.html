CSS Shapes: Advanced

Cutouts (evenodd), shaped borders (two-layer technique), and shape morphing animations.

Cutouts

Using data-cutout with the evenodd fill rule to punch shapes through elements.

Shaped Borders

Using data-border-shape with a ::before pseudo-element for colored shape outlines. Thickness controlled by --shape-border.

Shape Animation

Using data-morph for keyframe morphing between matching-point shapes. Respects prefers-reduced-motion.
