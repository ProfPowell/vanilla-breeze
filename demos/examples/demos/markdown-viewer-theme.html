# Brutalist Theme

This viewer has `data-theme="brutalist"` — bold, high-contrast styling.

- Monospace typography throughout
- Hard offset shadows
- Zero border radii

Editorial theme

# Editorial Theme

This viewer uses `data-theme="editorial"` — classic typographic elegance.

- Serif headings and refined spacing
- Warm-toned palette for long-form reading

Inherits theme from ancestor

This section has data-theme="brutalist". The viewer inherits it.

# Inherited Brutalist Theme

The component reads `data-theme` from the closest ancestor with `[data-theme]`.

No explicit theme attribute needed on the viewer itself.
