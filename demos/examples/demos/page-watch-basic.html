Page Watch demo

About this demo

Click the eye-icon button above to start watching this page. The button toggles to a “Watching” state and the notification panel’s Watching tab gets a count.

Open the bell icon and switch to the Watching tab to see the entry. Each row shows the page title, URL, and the time it was last checked. Use the x button to unwatch.

To simulate a content change: open devtools, paste localStorage.setItem('vb:watches:' + location.pathname, JSON.stringify({ data: { url: location.pathname, title: document.title, contentHash: 'stale', watchedAt: new Date().toISOString(), lastChecked: new Date().toISOString(), lastChanged: null, subscriptionId: null }, timestamp: Date.now() })), then reload. The Watching tab will show a “Changed” badge and a new notification of type watch appears in the All tab.

Server-side sync is opt-in. To enable it, add <meta name="vb-page-watch-server" content="true"> to your page’s <head> and have a /go/notify/subscribe endpoint reachable. Without it, the watch is purely client-side.
