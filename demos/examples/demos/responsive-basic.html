Auto-sizes (srcset already provided)

A ResizeObserver watches the rendered width and keeps sizes in sync, so the browser always picks the closest source for the actual layout. Loading and decoding defaults are added too.

Srcset shorthand from a width list

Drop the verbose srcset; let data-responsive="400,800,1200" synthesize it from src by inserting -{width} before the file extension.

Dev overlay (data-responsive-debug)

Add data-responsive-debug to inspect the chosen source — useful while tuning sizes or checking which width the browser picked.
