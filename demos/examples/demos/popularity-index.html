Popularity Index

The aggregated counterpart to <recently-visited>. Fetches a { pages: […] } payload from any compatible endpoint; the default is /api/analytics/popular.

Live (graceful degrade)

This site doesn't (yet) implement /api/analytics/popular, so the lens shows its unavailable-state message. That's the design — deploy without the endpoint and the panel quietly disappears with a friendly note.
