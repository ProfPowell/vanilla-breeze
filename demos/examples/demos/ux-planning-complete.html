Define User Stories

Sarah Chen view all project timelines in one dashboard I can quickly identify bottlenecks and report status
  • Dashboard loads within 2 seconds
  • All active projects visible without scrolling
  • Blocked tasks highlighted in red
 Sarah Chen export project reports as PDF I can share progress with stakeholders offline
  • PDF includes Gantt chart and status summary
  • Export completes in under 5 seconds
 Dev Patel receive real-time notifications when a blocker is raised I can unblock teammates without checking the dashboard
  • Push notification within 30 seconds of blocker
  • Notification links directly to the blocked task
 Dev Patel set API rate limits per client the system stays stable under heavy load
  • Default limit: 100 req/min per API key
  • 429 response with Retry-After header when exceeded

Build dashboard aggregation API

Fix timezone bug in scheduler

Events created in PST display at wrong time for EST users. Root cause: server stores naive datetime instead of UTC.

Migrate to Node 22 LTS

Node 20 reaches EOL in Q3. Upgrade runtime and verify all dependencies.

Evaluate WebSocket vs SSE for real-time

Compare latency, scaling, browser support, and infrastructure cost for real-time notifications.

Implement PDF export service

Headless Chrome service that renders dashboard as PDF with Gantt chart and status tables.
