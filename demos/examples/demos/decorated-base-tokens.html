Decorated Base Tokens

Specimen for three opt-in base decorations: animated borders via data-border-effect, themed scrollbars via data-scrollbar, and a cursor spotlight via data-spotlight. All defaults are inert — apply the attribute to opt in.

data-border-effect

Six CSS-only border animations. Add one keyword to any bordered element.

spin

Rotating conic-gradient through --color-primary/secondary/accent.

pulse

Box-shadow glow ring driven by --border-anim-pulse-*.

march

Marching-ants dashed outline from --border-anim-march-*.

hue-cycle

Border hue rotates via --border-anim-hue-*.

breathe

Offset ring scales in and out using --border-anim-breathe-*.

trace

On hover, each edge colours in sequence (--border-anim-trace-*). Hover me.

data-spotlight

Cursor-tracking radial spotlight. Set --spotlight-enabled:1 on the zone to activate the JS.

Move your cursor across this zone

A radial gradient follows the pointer, driven by --spotlight-x / --spotlight-y set on mousemove. Tune --spotlight-color and --spotlight-size per zone. Respects prefers-reduced-motion.

The spotlight script lazy-loads only when a [data-spotlight] element exists. Enable globally by setting --spotlight-enabled:1 on :root (e.g. a dramatic theme).
