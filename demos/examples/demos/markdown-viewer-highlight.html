Syntax Highlighting

The highlight attribute fires markdown-viewer:highlight per code block. Bring your own highlighter — here we use highlight.js.

# Code Examples

A JavaScript function:

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55
```

Some CSS:

```css
markdown-viewer {
  display: block;
  background: var(--color-surface);
  border-radius: var(--radius-m);
}

markdown-viewer[data-rendered] > pre {
  display: none;
}
```
