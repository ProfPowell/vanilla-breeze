Trade-offs

Aspect <progress-ring> <progress data-type="ring">
Author markup Manual ARIA + --progress Just value and max
Semantics Manual role="progressbar" Native (free)
JS update 3 properties to sync el.value = N (observer handles rest)
Without JS Broken (no visual) Standard progress bar (functional)
Center label Slotted (manual) Auto-generated from value
Indeterminate data-indeterminate Native (omit value)
API surface Separate element + docs One element, one doc page
