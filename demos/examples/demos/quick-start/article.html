Shipping a design system without a framework

By Jane Doe · April 20, 2026

Frameworks aren't the only way to scale. Semantic HTML, cascade layers, and a handful of custom elements can carry a brand further than you'd expect.

The pitch is familiar: pick a framework, adopt the ecosystem, then spend the next two years fighting it. What if we didn't?

"The most durable UI is the one that works before any JavaScript runs."

Three layers

HTML renders content. CSS provides design. JavaScript adds polish. Each layer is optional, each layer is testable, and the page still works when any one fails.

  • Semantic elements do the heavy lifting
  • Design tokens keep colors and spacing consistent
  • Web components fill the gaps — no build step required
