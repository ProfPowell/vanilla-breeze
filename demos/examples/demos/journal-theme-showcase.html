How it maps to Decorated Layers

A journaling theme isn't a skin — it's a stack. Each piece is a standard part wearing journal tokens.

  • Dot-grid paper
    data-theme="journal" — bg-wc surface concept as the theme's CSS page background
  • Washi tape
    <border-wc effect="washi"> — border-wc edge effect
  • Sketch border
    filter: var(--filter-rough-light) — VB rough-borders
  • Sticky note
    data-callout="sticky" — journal pack
  • Taped photo
    data-journal="photo" — journal pack
  • Rapid log
    data-journal="rapid-log" + li[data-bullet]
  • Habit tracker
    data-journal="tracker" + td[data-mark]
  • Highlighter & doodles
    <mark> + hr[data-ornament="doodle"]

Journal — part of the Vanilla Breeze extreme theme collection (theme + journal pack + border-wc + rough-borders).
