Shadows

The elevation ramp and the composable data-shadow effects.

Elevation ramp

Six themeable levels. Themes override --shadow-*, so each pack brings its own shadow language; the default theme uses a layered, photographic ramp.

--shadow-xs
--shadow-sm
--shadow-md
--shadow-lg
--shadow-xl
--shadow-2xl

data-shadow effects

Opt-in, composable, theme-aware. Each reads VB tokens, so they re-skin with the active theme.

tint

Accent-tinted drop shadow.

material

Hover to raise (md → xl).

glow

Soft accent halo.
