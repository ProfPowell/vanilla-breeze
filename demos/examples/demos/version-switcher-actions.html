version-switcher — Phase 2 actions (swap + diff)

The swap action fetches the picked version's HTML and replaces the [data-versioned] region in place (with View Transitions when supported). The diff action fetches both versions and renders a line-level diff inside a <change-set> mounted as a sibling of the versioned region — the change-set's tracking-view toggle then lets you flip between final / original / both.
