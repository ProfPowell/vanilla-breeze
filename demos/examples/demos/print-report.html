Key Findings

What Drove Adoption

  1. Zero-config theming — Teams could match brand guidelines by setting three custom properties instead of writing theme CSS from scratch.
  2. Accessibility built-in — Teams no longer needed to audit basic component accessibility; VB handles ARIA roles, focus management, and keyboard navigation.
  3. Reduced CSS maintenance — Average CSS bundle size dropped 40% across migrated projects.

Blockers

  • Two teams reported issues with existing !important overrides conflicting with VB’s cascade layers.
  • The mobile web team needed custom touch gestures not covered by existing components.
  • Marketing requested animation capabilities beyond VB’s current text effects.

Recommendations

  1. Set organization-wide target of 80% adoption by Q2 2026.
  2. Address mobile gesture gap with dedicated touch event component.
  3. Create migration playbook for teams with heavy !important usage.
  4. Expand animation system to cover marketing use cases (parallax, scroll-driven).
