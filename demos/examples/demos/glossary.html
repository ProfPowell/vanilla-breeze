A

attribute
A name–value pair in HTML markup that configures element behaviour or provides metadata. Example: data-provenance="ai-assisted".

C

change set
A group of related <ins> and <del> elements that can be toggled between tracking, final, and original views. See provenance.
custom element
An HTML element with a hyphenated tag name registered via customElements.define(). Provides encapsulated behaviour and styling.

D

design token
A named value (color, spacing, font size) stored as a CSS custom property. Tokens form the vocabulary of a design system.

H

HTML-first
A development approach that prioritises semantic HTML structure, using CSS for presentation and JavaScript only for progressive enhancement.

P

progressive enhancement
Building a baseline experience with HTML and CSS, then layering on JavaScript-powered features for capable browsers.
provenance
The origin and history of content: who created it, how it was made, and what changes it has undergone. See also: change set.

S

semantic HTML
Using HTML elements for their intended purpose (e.g., <nav> for navigation, <article> for self-contained content) rather than generic <div> and <span> elements.

W

web component
A custom element with encapsulated behaviour, typically using Shadow DOM, templates, and custom events.
