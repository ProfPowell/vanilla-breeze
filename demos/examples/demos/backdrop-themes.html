Page Background Tokens

The --page-bg-* token family controls rich body backgrounds. Each extreme theme overrides these tokens to create gradient backgrounds that complement the theme's palette.

Theme Gradient Showcase

  • --page-bg-gradient: linear-gradient(135deg, ...)

  • --page-bg-gradient: linear-gradient(180deg, ...)

  • --page-bg-gradient: linear-gradient(180deg, ...)

  • --page-bg-gradient: linear-gradient(to right, ...)

How It Works

The --page-bg-* tokens are defined in src/tokens/extensions/surfaces.css and consumed by the body rule in src/base/reset.css.

/* Token defaults (surfaces.css) */
--page-bg-type: solid;
--page-bg-color: var(--color-background, var(--color-surface));
--page-bg-gradient: none;

/* Body rule (reset.css) */
body {
  background-color: var(--page-bg-color);
  background-image: var(--page-bg-gradient, var(--page-bg-image, none));
}

Themes override these tokens in their :root[data-theme~="name"] block. The default theme uses --page-bg-type: solid, which produces no visual change from the existing var(--color-surface) background.
