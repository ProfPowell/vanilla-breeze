Chat Launcher Patterns

This page demonstrates chat integration patterns using existing VB primitives. Click the floating button in the bottom-right corner to open the chat dialog.

Floating Launcher

A fixed-position FAB button uses commandfor and command="show-modal" to open a <dialog> containing a full chat-window. No new elements needed.

Embedded with Sidebar

For always-visible chat panels, use <layout-sidebar> with chat history in the aside.
