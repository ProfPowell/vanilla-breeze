Basic Select

Standard <select> — auto-upgraded with no markup changes. The dropdown picker, options, checkmark, and chevron rotation are all CSS-controlled.

Grouped Select

<optgroup> headers render as styled section labels with dividers between groups.

Rich Content Select

A custom <button> trigger with <selectedcontent> mirrors the selected option. Options contain flag emoji and description text.

Browser Support

appearance: base-select requires Chrome 135+. VB wraps all styles in @supports (appearance: base-select), so non-supporting browsers fall back to the existing appearance: none + SVG chevron styling with no visual regression.

Detect support in JS:

if (CSS.supports('appearance', 'base-select')) { console.log('Customizable select supported'); }
