VB Callout Extension

Register the callout extension with marked to convert :::type blocks into styled callouts.

# Callout Blocks

The VB callout extension supports five types:

:::warning
**Careful!** This action cannot be undone. Make sure you have a backup before proceeding.
:::

:::info
The markdown-viewer component loads `marked` as its default parser. GFM is enabled by default.
:::

:::tip
Use the `highlight` attribute to fire per-block events for syntax highlighting integration.
:::

:::danger
Never use this component with untrusted user-generated markdown without enabling sanitization.
:::

:::note
The callout extension is a separate import — it adds zero bytes if you do not use it.
:::

## Regular Content

Paragraphs, **bold**, *italic*, and `code` still work as expected alongside callouts.
