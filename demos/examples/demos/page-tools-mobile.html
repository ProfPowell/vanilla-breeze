Mobile FAB Collapse

This demo uses a high breakpoint (80rem) so the FAB collapse is always active. Look for the floating action button in the bottom-right corner. Tap it to open a popover containing the tools.

When the viewport expands above the breakpoint, the tools return to their original inline position. This is a DOM move, not a clone, so event listeners on tool children are preserved across the transition.

The popover uses native popover behavior for light dismiss — clicking outside the panel or pressing Escape closes it. Focus returns to the FAB button when the popover is dismissed.
