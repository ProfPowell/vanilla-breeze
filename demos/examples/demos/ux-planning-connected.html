Persona

The persona anchors everything. The empathy map, journey, and stories all reference Sarah.

Sarah Chen

I need tools that help me stay organized across multiple projects without context switching.

Sarah has 8 years of experience in product management, starting at startups and now leading a team at a Series C company. She manages 3 product teams and oversees a roadmap with 50+ features.

  • Streamline cross-team communication
  • Reduce time spent in status meetings
  • Have a single source of truth for project progress
  • Too many disconnected tools (Jira, Slack, Notion, Figma)
  • Spending 60% of time in meetings instead of strategy
  • Unclear ownership across distributed teams
  • Checks Slack 30+ times per day
  • Prefers async communication when possible
  • Reviews metrics dashboard every morning

Unified timeline dashboard

Product Manager see all project timelines in a unified dashboard I can identify bottlenecks without switching tools
  • Dashboard loads within 2 seconds for up to 50 projects
  • All active projects display with color-coded status
  • Timeline view supports daily to quarterly zoom

Automated status summaries

Product Manager receive automated weekly status summaries I can skip manual report compilation
  • Summary email sent every Monday at 9am
  • Includes changes since last summary

Custom alert thresholds

Product Manager set custom alert thresholds per project I only get notified about things that matter

Stakeholder dashboard sharing

Product Manager share a read-only dashboard with stakeholders executives can self-serve instead of asking me
