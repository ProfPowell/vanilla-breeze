Understanding Page Tools

The toolbar below aggregates page-level utilities. Each direct child is treated as a tool automatically — no registration needed.

Three positioning strategies are available: sticky for sidebar placement, fixed for viewport-edge pinning, and inline (shown here) which flows with the document.

On small viewports, the toolbar can collapse into a floating action button (FAB) that opens a native popover. The breakpoint is configurable. This demo disables the collapse to keep the tools always visible.
