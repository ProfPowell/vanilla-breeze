What is progressive enhancement?

A strategy that starts with a baseline of usable functionality, then layers on enhancements for browsers that support them. The details element itself is a perfect example.

Can I use this without JavaScript?

Yes. The name attribute creates an exclusive accordion entirely with HTML. Opening one panel automatically closes the other panels that share the same name.

Which browsers support the name attribute?

Chrome 120+, Safari 17.2+, and Firefox 130+ all support the name attribute on details elements.
