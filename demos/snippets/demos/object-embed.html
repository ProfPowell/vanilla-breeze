<object> — SVG embed

<object> — HTML embed (with fallback)

<embed> — SVG (no fallback)

<embed> is a void element — no fallback content possible. Prefer <object> or semantic alternatives.

Prefer modern alternatives

  • <img> for images (including SVG)
  • <svg> for inline vector graphics
  • <video> for video content
  • <audio> for audio content
  • <iframe> for embedded pages
