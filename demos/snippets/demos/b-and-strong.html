Bold and Strong Usage

Basic bold

Bold text draws attention without conveying importance. Strong text conveys importance or urgency.

Bold in code (highlight background)

In code contexts, bold gets a highlight instead of just weight:

const result = calculateTotal(items);
console.log(result);

Combined bold + italic

Nested bold and italic combine properly: bold italic via b+i and bold italic via strong+em.

Use cases for <b>

A product name in a review. A keyword in a paragraph summary. The lead sentence of a news article.
