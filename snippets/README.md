# Vanilla Breeze Snippets

Ready-to-use code snippets for building with Vanilla Breeze.

## Directory Structure

```
snippets/
├── html/           # HTML page patterns
├── css/            # CSS token and style snippets
└── components/     # Component usage examples
```

## HTML Snippets

| File | Description |
|------|-------------|
| `page-shell.html` | Base HTML5 document structure |
| `nav-tree.html` | Tree navigation with details/summary |
| `article-layout.html` | Content page with sidebar |
| `form-basic.html` | Accessible form with form-field |
| `card-grid.html` | Responsive card layout |
| `hero-section.html` | Hero section patterns |

## CSS Snippets

| File | Description |
|------|-------------|
| `tokens-starter.css` | Minimal token setup |
| `layer-setup.css` | CSS @layer configuration |
| `dark-mode-toggle.css` | Theme switching styles |
| `responsive-grid.css` | Grid layout patterns |
| `typography-scale.css` | Type hierarchy |

## Component Snippets

| File | Description |
|------|-------------|
| `accordion-basic.html` | Accordion patterns |
| `tabs-horizontal.html` | Tab interface examples |
| `toast-notification.html` | Toast notification usage |
| `modal-dialog.html` | Native dialog patterns |
| `dropdown-menu.html` | Dropdown menu examples |

## Usage

Copy the relevant snippet into your project and customize as needed.
All snippets use Vanilla Breeze tokens and follow project conventions.
