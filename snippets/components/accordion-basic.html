Frequently Asked Questions

How do I get started?

Install the package via npm and import the styles into your project.

npm install vanilla-breeze
Is JavaScript required?

No! The accordion works without JavaScript using native HTML details/summary. JavaScript enhances the experience with keyboard navigation and ARIA support.

Can I customize the appearance?

Yes, use CSS custom properties to adjust colors, spacing, and animations.
