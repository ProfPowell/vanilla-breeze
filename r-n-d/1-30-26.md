## Demo Details

### 1. Motion Tokens
Exploring expanded motion capabilities:
- **Hover effects**: lift, glow, scale, slide, border, tilt
- **Transition presets**: snappy, smooth, bouncy, elastic
- **Enter/exit animations**: fade, slide-up, slide-down, scale, pop
- **Staggered animations**: adjustable delay with CSS custom properties
- **View Transitions API**: card expand/collapse demo
- **Theme personalities**: different motion feels per theme
- **Reduced motion**: toggle and system preference support

**Proposed tokens**:
```css
--motion-hover-transform: translateY(-2px);
--motion-stagger-delay: 50ms;
--motion-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 2. Backgrounds & Surfaces
Exploring surface treatments:
- **Gradients**: linear, radial, conic, mesh, animated, layered
- **Textures**: noise, dots, grid, lines, grain (via SVG data URIs)
- **Glassmorphism**: adjustable backdrop-filter blur
- **Nested surfaces**: demonstrating VB surface token hierarchy
- **Page backgrounds**: solid, gradient, radial, noise options

**Proposed tokens**:
```css
--surface-texture: none | noise | grain | dots | lines;
--surface-texture-opacity: 0.05;
--glass-blur: 8px;
```

### 3. Rough Borders
Exploring non-standard border techniques:
- **SVG filters**: feTurbulence + feDisplacementMap (light/medium/heavy)
- **Border-image**: hand-drawn SVG patterns (sketch, rough, marker)
- **Mask-image**: torn edges, wavy edges
- **CSS gradients**: dashed borders, dot borders
- **Box shadows**: sketch effect, hard offset (brutalist)
- **Pseudo-elements**: corner accents, tape effect

**Proposed tokens**:
```css
--border-style: solid | dashed | rough | sketchy | none;
--border-roughness: 0 | 1 | 2 | 3;
```

### 4. Custom Fonts
Testing external font integration:
- **Display fonts**: Playfair Display, Abril Fatface, Bebas Neue
- **Body fonts**: Inter, Source Sans 3, Literata
- **Mono fonts**: JetBrains Mono, Fira Code
- **Font pairings**: 4 example combinations
- **Font loading**: status indicator, performance metrics
- **font-display strategies**: swap, block, fallback, optional

**Proposed tokens**:
```css
--font-display: "Playfair Display", serif;
--font-body-external: "Inter", var(--font-sans);
```

### 5. Sound Effects (Experimental)
Exploring audio feedback:
- **Web Audio API**: generated sounds (no audio files)
- **Sound types**: click, success, error, warning, info, toggle, hover
- **Controls**: master toggle, volume slider
- **Settings**: per-sound-type enable/disable
- **Persistence**: localStorage for user preferences

**Proposed token**:
```css
--sound-enabled: 1;  /* Read by JS */
```

## Technical Notes

### Serving the Demos
The demos are static HTML files. They're copied to `r-n-d/zen-garden/` which is symlinked from `public/r-n-d`.

**CSS Path**: All demos use `/cdn/vanilla-breeze.css` for the core styles.

**Local testing**: Run a server from the `dist` folder:
```bash
cd dist && python3 -m http.server 8888
# Then visit http://localhost:8888/r-n-d/theme-extensions/
```

### File Locations
- Source files: `r-n-d/theme-extensions/`, `r-n-d/theme-picker/`, `r-n-d/theme-lab/`
- Served via symlink: `r-n-d/zen-garden/theme-extensions/`, etc.
- Built output: `dist/r-n-d/theme-extensions/`, etc.
