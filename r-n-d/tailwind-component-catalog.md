# Tailwind Component Catalog for Vanilla Breeze

Comprehensive catalog of Tailwind CSS ecosystem components for porting to semantic HTML/CSS framework.

**Priority Levels:**
- P0: Essential core components (buttons, forms, layout)
- P1: Common UI patterns (navigation, cards, modals)
- P2: Enhanced functionality (datepickers, charts)
- P3: Specialized/advanced (3D effects, complex interactions)

## IMPLEMENTATION NOTES

### Phase 1 (P0 - Core Essentials)
Focus on fundamental components that form the building blocks:
- Forms (inputs, selects, checkboxes, radio, toggles)
- Buttons and basic actions
- Layout primitives (containers, cards)
- Tables and basic lists
- Typography system
- Avatars and badges
- Basic navigation (navbar)
- Alerts

### Phase 2 (P1 - Common Patterns)
Add frequently used UI patterns:
- All navigation components (tabs, breadcrumbs, sidebar, pagination)
- Modals and notifications
- Progress indicators and spinners
- Extended form elements
- Button groups and dropdowns
- Card variants and list types
- Marketing sections (hero, features, CTA, pricing)

### Phase 3 (P2 - Enhanced Functionality)
Implement specialized components:
- Datepicker and calendar
- Command palette
- Drawers and popovers
- Skeleton loaders
- Carousels and accordions
- All feedback components
- Marketing page elements
- Ecommerce components

### Phase 4 (P3 - Advanced/Specialized)
Polish and specialized use cases:
- Speed dial
- Bento grids
- Complete page templates
- Complex interactions
- Video components
- Advanced ecommerce features

### Key Principles for Vanilla Breeze
1. **Semantic HTML first** - Use proper elements (button, select, details, etc.)
2. **No JavaScript dependencies** - Progressive enhancement only
3. **CSS custom properties** - Theming through variables
4. **Accessible by default** - Proper ARIA, keyboard navigation
5. **Minimal markup** - Clean, readable HTML structure
6. **Platform features** - Use :has(), :focus-within, container queries, etc.
7. **Component classes** - Semantic names like daisyUI, not utility classes
