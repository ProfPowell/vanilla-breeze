Theme Picker Variants

Exploring different UI patterns for theme selection: icon grids, rich previews, dropdowns, and sidebar panels.

Variant A: Icon Grid Picker

Compact icon/swatch grid that fits in headers. Tooltip on hover shows theme name. Best for limited space.

Variant B: Rich Preview List

Full card per theme with color swatches, typography preview, and description. Scrollable panel for many themes.

Variant C: Dropdown Select

Grouped dropdown with categories. Good for many themes without taking up much space.
